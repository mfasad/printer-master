Как изменить принтер по умолчанию в Windows 10: все способы

Изменение принтера по умолчанию в Windows 10 — задача, с которой сталкивается каждый пользователь, подключающий несколько печатных устройств к одному компьютеру. Система автоматически назначает основным последний использованный принтер, но это не всегда удобно: например, если вам нужно регулярно печатать на Canon LBP2900B для документов, а фотографии отправлять на Epson EcoTank L3250. В этой статье разберём все актуальные способы смены принтера по умолчанию, включая скрытые настройки и решение типичных проблем.

Особенность Windows 10 (в отличие от предыдущих версий) — функция "Управление принтерами по умолчанию", которая может автоматически переключать устройство в зависимости от физического расположения или типа задачи. Это полезно в офисах, но часто мешает дома. Мы покажем, как отключить эту опцию и зафиксировать нужный принтер вручную. Также вы узнаете, что делать, если система игнорирует ваши настройки или принтер исчезает из списка после перезагрузки.

Почему Windows 10 автоматически меняет принтер по умолчанию?

Начиная с версии Windows 10 Anniversary Update (1607), Microsoft добавила функцию автоматического управления принтерами. Система анализирует:

  • 📍 Физическое расположение: если вы подключены к корпоративной сети, Windows может выбирать сетевой принтер офиса вместо домашнего.
  • 🖨️ Последний использованный принтер: система запоминает, на какое устройство вы отправляли задачу в последний раз.
  • 📄 Тип документа: для фотографий может выбираться принтер с поддержкой borderless printing, а для текста — монохромное устройство.

Эта логика работает через службу Print Management и настройки групповой политики (в корпоративных редакциях Windows). Для домашних пользователей такое поведение часто становится неудобным — например, если основной принтер Brother HL-L2350DW suddenly сменяется на виртуальный Microsoft Print to PDF после создания PDF-файла.

Способ 1: Изменение через «Параметры Windows» (самый простой)

Это основной метод, который работает во всех редакциях Windows 10, включая Home и Pro. Инструкция актуальна для сборок 1809 и новее:

  1. Откройте меню Пуск → Параметры (⚙️) → Устройства → Принтеры и сканеры.
  2. В списке устройств найдите нужный принтер (например, HP LaserJet Pro M404n) и кликните на него левой кнопкой мыши.
  3. Нажмите кнопку Управление.
  4. В новом окне выберите Установить по умолчанию.

Если кнопка Установить по умолчанию неактивна (серого цвета), это означает, что:

  • 🔌 Принтер отключён от сети или компьютера (проверьте кабель USB или подключение по Wi-Fi).
  • 🚫 У устройства нет драйверов (в диспетчере задач может отображаться как "Неопознанное устройство").
  • 🔒 Настройки заблокированы групповой политикой (актуально для рабочих ПК в офисах).

Способ 2: Через «Панель управления» (классический метод)

Этот способ подходит для пользователей, привыкших к интерфейсу старых версий Windows, а также в случаях, когда новый интерфейс Параметров работает некорректно. Пошаговая инструкция:

  1. Откройте Панель управления (можно найти через поиск в меню Пуск).
  2. Перейдите в раздел Устройства и принтеры (в режиме отображения "Крупные значки" или "Мелкие значки").
  3. Найдите нужный принтер в списке, кликните по нему правой кнопкой мыши и выберите Использовать по умолчанию.

Если принтер отсутствует в списке, попробуйте:

  • 🔄 Обновить список устройств кнопкой Обновить в верхней панели.
  • 📥 Переустановить принтер: удалите его из списка, затем добавьте заново через Добавить принтер.
  • 🖼️ Проверьте, не скрыт ли принтер в разделе "Невидимые устройства" (актуально для сетевых принтеров).
💡

Если принтер отображается с восклицательным знаком, попробуйте удалить его, затем подключить заново с установкой драйверов вручную с официального сайта производителя (например, support.hp.com или canon.ru).

Способ 3: Через командную строку (для опытных пользователей)

Этот метод полезен, если графический интерфейс недоступен (например, при удалённом подключении через RDP) или нужно автоматизировать процесс для нескольких компьютеров. Используем утилиту rundll32:

rundll32 printui.dll,PrintUIEntry /y /n "Имя принтера"

Замените Имя принтера на фактическое название вашего устройства (например, Brother DCP-T720DW). Чтобы узнать точное имя:

  1. Откройте Параметры → Устройства → Принтеры и сканеры.
  2. Запомните или скопируйте название принтера из списка (оно чувствительно к регистру!).

Для проверки текущего принтера по умолчанию используйте команду:

wmic printer get name,default

В ответе ищите строку TRUE — это и есть текущее устройство по умолчанию.

Как сбросить все настройки принтеров через командную строку?

Используйте команду printui /s /t2, чтобы открыть окно настройки принтеров, где можно удалить все устройства и драйверы. Внимание: после этого потребуется повторная установка принтеров!

Как отключить автоматическое изменение принтера по умолчанию?

Функция "Разрешить Windows управлять принтером по умолчанию" включена по умолчанию. Чтобы её отключить:

  1. Перейдите в Параметры → Устройства → Принтеры и сканеры.
  2. Снимите флажок напротив опции Разрешить Windows управлять принтером по умолчанию.
  3. Выберите нужный принтер вручную (как описано в Способе 1).

В корпоративных сетях эта опция может быть заблокирована через Групповую политику. Чтобы проверить:

  1. Нажмите Win + R, введите gpedit.msc и нажмите Enter.
  2. Перейдите по пути: Конфигурация пользователя → Административные шаблоны → Панель управления → Принтеры.
  3. Проверьте параметр Запретить изменение принтера по умолчанию. Если он включён, обратитесь к администратору сети.
💡

Отключение автоматического управления принтерами — единственный способ зафиксировать выбранное устройство навсегда. Без этого Windows будет сбрасывать настройки после каждого документа.

Что делать, если принтер по умолчанию не сохраняется?

Если после перезагрузки или печати документа принтер снова сбрасывается, проверьте следующие моменты:

⚠️ Внимание: Если вы используете OneDrive или другие облачные сервисы для синхронизации настроек, они могут перезаписывать ваши изменения. Отключите синхронизацию параметров принтера в настройках учётной записи Microsoft.
  • 🔄 Очистите очередь печати:
    1. Откройте Панель управления → Устройства и принтеры.
    2. Кликните правой кнопкой по принтеру → Просмотр очереди печати.
    3. В меню Принтер выберите Очистить очередь печати.
  • 🛠️ Переустановите драйверы:
    1. Удалите принтер через Параметры → Устройства → Принтеры и сканеры.
    2. Скачайте свежие драйверы с сайта производителя (например, для Xerox Phaser 3020 — с xerox.com).
    3. Установите принтер заново, выбрав опцию "Добавить локальный или сетевой принтер с параметрами вручную".
  • 🔧 Проверьте службу печати:
    1. Нажмите Win + R, введите services.msc.
    2. Найдите службу Диспетчер печати (Print Spooler).
    3. Если она остановлена, запустите её и установите Тип запуска: Автоматически.

    • Если принтер по умолчанию сбрасывается только для конкретных программ (например, Microsoft Word или Adobe Acrobat), проблема может быть в настройках самого приложения. Проверьте параметры печати внутри программы — некоторые ПО (например, AutoCAD) игнорируют системные настройки принтера.

    FAQ: Частые вопросы по смене принтера по умолчанию

    Можно ли назначить принтер по умолчанию для конкретной программы?

    Да, некоторые программы (например, Adobe Photoshop или CorelDRAW) позволяют сохранять настройки принтера для каждого проекта. Для этого:

    1. Откройте диалог печати в программе (Ctrl + P).
    2. Выберите нужный принтер и настройте параметры.
    3. Сохраните пресет (если есть такая опция) или экспортируйте профиль печати.

    В Microsoft Office настройки принтера сохраняются вместе с документом, но не глобально для всей программы.

    Почему после обновления Windows 10 принтер по умолчанию сбрасывается?

    Это связано с тем, что крупные обновления (например, Windows 10 22H2) сбрасывают некоторые пользовательские настройки. Чтобы избежать проблемы:

    1. Перед обновлением зафиксируйте текущий принтер по умолчанию (отключите автоматическое управление).
    2. После обновления переустановите драйверы принтера.
    3. Проверьте, не появились ли дублирующиеся записи принтера в списке устройств.
    ⚠️ Внимание: Если после обновления принтер исчез из списка, попробуйте подключить его по IP-адресу (для сетевых устройств) или через USB в другом порту.
    Как назначить сетевой принтер по умолчанию?

    Для сетевого принтера (например, Kyocera Ecosys P3045dn) сначала убедитесь, что:

    • Принтер подключён к той же сети, что и компьютер.
    • На принтере включён протокол SMB или LPD (проверьте в веб-интерфейсе устройства).
    • Брандмауэр Windows не блокирует порт 9100 (для RAW-печати).

    Затем добавьте принтер через Параметры → Устройства → Принтеры и сканеры → Добавить принтер или сканер, выбрав опцию "Нужный принтер отсутствует в списке" и указав IP-адрес.

    Можно ли назначить виртуальный принтер (например, PDF) по умолчанию?

    Да, виртуальные принтеры (например, Microsoft Print to PDF, CutePDF или doPDF) можно сделать основными. Однако учтите:

    • Некоторые программы (например, 1С:Предприятие) могут некорректно работать с виртуальными принтерами по умолчанию.
    • При печати из браузера (Chrome, Edge) может автоматически выбираться последний физический принтер.

    Чтобы назначить виртуальный принтер, используйте любой из описанных выше способов (через Параметры или Панель управления).
