Как посмотреть принтер по умолчанию: 5 проверенных способов для Windows и Mac

Когда вы отправляете документ на печать, операционная система автоматически выбирает принтер по умолчанию — устройство, которое будет использоваться для вывода файла без дополнительных подтверждений. Однако не всегда этот выбор оказывается правильным: например, после подключения нового МФУ или сброса настроек системы. В результате документы могут уходить на не тот принтер, тратить бумагу впустую или вообще не печататься, если выбранное устройство отключено.

В этой статье вы найдёте актуальные способы проверки принтера по умолчанию для всех версий Windows (от 7 до 11) и macOS (включая Ventura и Sonoma), а также инструкции по изменению настроек и устранению типичных ошибок. Мы разберём как стандартные методы через графический интерфейс, так и альтернативные — через командную строку и PowerShell. Отдельно остановимся на нюансах для сетевых принтеров и виртуальных устройств (например, Microsoft Print to PDF).

1. Как узнать принтер по умолчанию в Windows 10 и 11

В современных версиях Windows процесс максимально упрощён. Самый быстрый способ — использовать панель управления принтерами, которая доступна из системного трея или через меню «Параметры».

Следуйте пошаговой инструкции:

  1. Откройте меню Пуск и выберите Параметры (значок шестерёнки).
  2. Перейдите в раздел Устройства → Принтеры и сканеры.
  3. В списке устройств найдите принтер с пометкой «По умолчанию» (или галочкой зелёного цвета в Windows 11).

Если у вас Windows 10 версии 1809 и новее, система может автоматически назначать принтер по умолчанию в зависимости от вашего местоположения (функция «Разрешить Windows управлять принтером по умолчанию»). Чтобы отключить эту опцию, прокрутите страницу вниз и снимите соответствующий флажок.

💡

В Windows 11 список принтеров можно открыть быстрее: нажмите Win + R, введите ms-settings:printers и нажмите Enter.

2. Альтернативный метод: через панель управления

Классическая панель управления остаётся актуальной даже в Windows 11, особенно для пользователей, привыкших к старому интерфейсу. Этот способ работает и в Windows 7, где современного меню «Параметры» просто нет.

Инструкция:

  1. Нажмите Win + R, введите control printers и нажмите Enter.
  2. В открывшемся окне найдите принтер с зелёной галочкой — это и есть устройство по умолчанию.
  3. Чтобы изменить выбор, кликните правой кнопкой по нужному принтеру и выберите Использовать по умолчанию.

В Windows 7 интерфейс может немного отличаться: вместо зелёной галочки используется жирный шрифт названия принтера и надпись (По умолчанию) справа. Если принтеров несколько, а галочки нет ни у одного, значит система не смогла определить устройство автоматически — его нужно назначить вручную.

3. Просмотр через командную строку (CMD) и PowerShell

Для опытных пользователей или администраторов, которым нужно быстро получить информацию о принтерах на множестве машин, подойдут консольные команды. Этот метод универсален и работает во всех версиях Windows.

Способ 1: через CMD (командную строку):

wmic printer get name,default

В результате вы увидите список всех принтеров с колонкой Default, где TRUE обозначает устройство по умолчанию.

Способ 2: через PowerShell (более современный вариант):

Get-Printer | Select-Object Name, ComputerName, Shared, Type, DriverName | Format-Table -AutoSize

Здесь принтер по умолчанию будет отмечен в столбце Type значением Connection: Default.

Как сохранить список принтеров в файл?

Чтобы экспортировать данные в текстовый файл, добавьте к команде PowerShell параметр > C:\printers.txt. Например:

Get-Printer | Select-Object Name, Default | Out-File -FilePath C:\printers_list.txt

Файл появится на диске C:.

Эти команды полезны для диагностики проблем — например, если принтер по умолчанию внезапно сбросился после обновления системы или подключения нового устройства. Также они пригодятся системным администраторам для массовой настройки принтеров в корпоративной сети.

4. Особенности для macOS (Ventura, Sonoma и старые версии)

На компьютерах Mac процесс проверки принтера по умолчанию несколько отличается от Windows, но не менее интуитивен. Apple использует единый центр управления печатью — «Очередь печати».

Пошаговая инструкция:

  1. Откройте меню Apple (логотип в левом верхнем углу) и выберите Системные настройки.
  2. Перейдите в раздел Притерры и сканеры.
  3. В списке устройств слева найдите принтер с пометкой «По умолчанию» (или Default в английской версии).
  4. Чтобы изменить выбор, кликните по нужному принтеру правой кнопкой и выберите Сделать принтером по умолчанию.

В macOS Sonoma (2023 года) интерфейс немного обновился: теперь принтеры отображаются в виде плиток, а пометка «По умолчанию» стала менее заметной — она появляется только при наведении курсора. Если принтер не определяется автоматически, попробуйте добавить его вручную через кнопку «+» внизу списка.

Важно: в macOS принтер по умолчанию может сбрасываться после обновления системы или при подключении к другой сети (например, при переезде с работы домой). Чтобы этого избежать, закрепите настройки вручную.

5. Проблемы с принтером по умолчанию и их решения

Иногда система некорректно определяет принтер по умолчанию или вообще не сохраняет изменения. Рассмотрим типовые ошибки и способы их устранения:

  • 🔄 Принтер сбрасывается после перезагрузки: отключите функцию «Разрешить Windows управлять принтером по умолчанию» в параметрах устройства.
  • ❌ Нет пометки «По умолчанию» ни у одного принтера: назначьте устройство вручную (клик правой кнопкой → «Использовать по умолчанию»).
  • 🖨️ Документы уходят не на тот принтер: проверьте, не установлен ли в программе (например, Word или Excel) свой принтер по умолчанию, отличный от системного.
  • 🌐 Сетевой принтер не сохраняется как основной: обновите драйверы или переподключите устройство через IP-адрес.

Если принтер по умолчанию внезапно изменился после подключения к другой сети (например, через VPN или при смене Wi-Fi), это связано с функцией геолокации в Windows 10/11. Чтобы отключить её:

  1. Перейдите в Параметры → Устройства → Принтеры и сканеры.
  2. Отключите опцию Разрешить Windows управлять принтером по умолчанию.
💡

Если принтер по умолчанию сбрасывается после обновления Windows, попробуйте переустановить его драйверы через «Диспетчер устройств».

Для macOS типичная проблема — «зависание» очереди печати. Чтобы сбросить её:

  1. Откройте Очередь печати (значок принтера в Dock или через Системные настройки).
  2. Нажмите Очистить очередь (или Reset printing system в английской версии).
  3. Перезагрузите компьютер и назначьте принтер по умолчанию заново.

6. Как проверить принтер по умолчанию для конкретной программы

Некоторые приложения (например, Adobe Acrobat, Microsoft Office или AutoCAD) могут игнорировать системные настройки и использовать свой собственный принтер по умолчанию. Это часто становится причиной путаницы, когда документы из одного софта печатаются на одном устройстве, а из другого — на другом.

Как проверить и изменить настройки:

Программа Путь к настройкам печати Как изменить принтер
Microsoft Word/Excel Файл → Печать В выпадающем меню принтеров выберите нужное устройство и нажмите Установить как принтер по умолчанию (если доступно).
Adobe Acrobat Reader Файл → Печать или Ctrl+P Выберите принтер в списке и поставьте галочку Сохранить как принтер по умолчанию.
Google Chrome Меню (⋮) → Печать или Ctrl+P Нажмите Изменить рядом с полем «Назначение» и выберите принтер. В Chrome нет опции сохранения по умолчанию — он использует системные настройки.
AutoCAD Файл → Печать или команда PLOT В окне печати выберите принтер в разделе Имя принтера и нажмите Сохранить изменения.

Обратите внимание: в браузерах (Chrome, Firefox, Edge) обычно нет отдельной настройки принтера по умолчанию — они наследуют её из системы. Однако некоторые веб-приложения (например, Google Docs) могут сохранять последний выбранный принтер.

💡

В Photoshop принтер по умолчанию задаётся в меню Файл → Печать, но сохраняется только для текущего сеанса. Чтобы зафиксировать настройки, используйте Предварительные настройки печати (Print Settings).

7. Виртуальные принтеры (PDF, XPS) и их настройки

Помимо физических устройств, в системе могут быть установлены виртуальные принтеры, такие как:

  • 📄 Microsoft Print to PDF — сохраняет документы в формат PDF.
  • 📑 Microsoft XPS Document Writer — создаёт файлы XPS.
  • 🖼️ OneNote — отправляет «печать» в блокнот OneNote.
  • ☁️ Google Cloud Print (устаревший сервис, но ещё встречается в старых системах).

Эти принтеры тоже могут становиться устройствами по умолчанию, что иногда приводит к путанице. Например, если вы ожидаете распечатку на бумаге, а документ сохраняется в PDF. Чтобы избежать этого:

  1. Проверьте список принтеров в Параметрах (как описано в разделе 1).
  2. Если виртуальный принтер установлен по умолчанию, смените его на физическое устройство.
  3. В некоторых программах (например, Word) можно отключить отображение виртуальных принтеров в списке печати.

В Windows 11 виртуальный принтер Microsoft Print to PDF часто устанавливается по умолчанию после чистой установки системы или сброса настроек. Это сделано для удобства пользователей, которые чаще сохраняют документы в PDF, чем печатают на бумаге. Однако для офисных задач такой выбор может быть неудобен.

8. Автоматизация: как всегда печатать на одном принтере

Если вам надоело постоянно менять принтер по умолчанию или вы работаете в сети с несколькими устройствами, можно зафиксировать настройки с помощью скриптов или групповой политики (для корпоративных пользователей).

Способ 1: через PowerShell (для одного компьютера):

Set-Printer -Name "Имя_вашего_принтера" -Default

Замените Имя_вашего_принтера на фактическое название устройства (можно скопировать из списка в Get-Printer).

Способ 2: через редактор групповой политики (для администрирования сети):

  1. Нажмите Win + R, введите gpedit.msc и нажмите Enter.
  2. Перейдите по пути: Конфигурация пользователя → Административные шаблоны → Панель управления → Принтеры.
  3. Активируйте политику Запретить добавление принтеров или Всегда устанавливать конкретный принтер по умолчанию.

Для домашних пользователей проще всего отключить автоматическое управление принтерами в параметрах Windows, как описано в разделе 5. Это предотвратит случайные изменения настроек.

💡

Используйте PowerShell-скрипты для автоматического назначения принтера по умолчанию при запуске системы. Пример скрипта можно добавить в Автозагрузку.

⚠️ Внимание: В корпоративных сетях настройки принтеров по умолчанию могут управляться администратором через Active Directory или Group Policy. В этом случае изменения на локальном компьютере не сохранятся после перезагрузки.

FAQ: Частые вопросы о принтере по умолчанию

Можно ли сделать принтер по умолчанию для конкретной программы, не меняя системные настройки?

Да, некоторые программы (например, Adobe Acrobat или AutoCAD) позволяют сохранять свой принтер по умолчанию независимо от системы. Для этого откройте настройки печати в самой программе и найдите опцию типа Сохранить как принтер по умолчанию или Use as Default.

Почему после подключения к другой сети принтер по умолчанию сбрасывается?

В Windows 10 и 11 включена функция Разрешить Windows управлять принтером по умолчанию, которая выбирает устройство в зависимости от вашего местоположения (определяется по сети). Чтобы отключить её, перейдите в Параметры → Устройства → Принтеры и сканеры и снимите соответствующий флажок.

Как узнать принтер по умолчанию в Linux (Ubuntu, Mint)?

В большинстве дистрибутивов Linux используйте команду в терминале: 

lpstat -d

Чтобы изменить принтер, выполните:

lpoptions -d имя_принтера

Для графического интерфейса откройте Настройки → Принтеры и найдите устройство с пометкой Default.

Можно ли сделать сетевой принтер устройством по умолчанию для всех пользователей на компьютере?

Да, но потребуются права администратора. В Windows используйте PowerShell с повышенными правами: 

Set-Printer -Name "Имя_сетевого_принтера" -Shared -Published -Default

Для macOS настройте принтер в Системных настройках и примените изменения ко всем учётным записям через Параметры совместного доступа.

Почему принтер по умолчанию не сохраняется после обновления Windows?

Это известная проблема, связанная с сбросом некоторых пользовательских настроек при крупных обновлениях (например, при переходе с Windows 10 на 11). Решение:

  1. Переустановите драйвер принтера через Диспетчер устройств.
  2. Назначьте принтер по умолчанию заново.
  3. Отключите автоматическое управление принтерами в параметрах.

Если проблема повторяется, используйте скрипт для автоматического назначения принтера (см. раздел 8).

Теперь вы знаете, как проверить и изменить принтер по умолчанию на любом устройстве. Если после применения этих методов проблема сохраняется, проверьте драйверы принтера и сетевые настройки — часто ошибки печати связаны именно с ними.
