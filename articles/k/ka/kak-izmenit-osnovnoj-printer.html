Как поменять принтер по умолчанию: 5 способов для Windows и macOS

Смена основного принтера — задача, с которой сталкивается каждый пользователь, подключающий к компьютеру несколько печатных устройств. По умолчанию система автоматически назначает приоритет последнему добавленному принтеру или тому, который чаще используется, но это не всегда удобно. Например, в офисе может потребоваться быстро переключаться между цветным Canon PIXMA для презентаций и монохромным Brother HL-L2350DW для внутренних документов, а дома — между HP DeskJet для фотографий и Epson EcoTank для многостраничных отчётов.

В этой статье вы найдёте актуальные способы изменения принтера по умолчанию для всех современных версий операционных систем, включая редкие сценарии (например, через PowerShell или групповую политику в корпоративных сетях). Мы также разберём типичные ошибки, из-за которых новый принтер не сохраняется как основной, и дадим рекомендации по оптимизации процесса печати для разных задач.

Почему система автоматически меняет принтер по умолчанию?

Начиная с Windows 10 (версия 1803) и macOS Mojave, операционные системы получили функцию «Управление принтерами по умолчанию», которая анализирует поведение пользователя. Алгоритм работает так:

  • 📍 Если вы печатаете на одном устройстве чаще других, система предлагает сделать его основным.
  • 🔄 При подключении к новой сети (например, в офисе или кафе) Windows может временно сменить принтер на сетевой.
  • 🖨️ После установки нового драйвера или перезагрузки ПК иногда сбрасываются настройки.

Эта «умная» функция часто мешает: например, если вы редко печатаете, но нужно, чтобы основным оставался конкретный принтер. Отключить её можно в настройках — об этом расскажем в соответствующем разделе.

⚠️ Внимание: В корпоративных сетях с доменными политиками настройки принтера по умолчанию могут блокироваться администратором. В этом случае изменения вступят в силу только после перезагрузки или входа в систему.

Способ 1: Изменение принтера по умолчанию в Windows 10 и 11 через «Параметры»

Самый простой метод — использовать графический интерфейс. Подходит для большинства пользователей и не требует прав администратора.

  1. Откройте меню Пуск и выберите Параметры → Устройства → Принтеры и сканеры.
  2. В списке найдите нужный принтер (например, Epson L3250) и кликните по нему левой кнопкой мыши.
  3. Нажмите кнопку «Управление», затем — «Назначить по умолчанию».

Если кнопка неактивна, проверьте:

  • 🔌 Подключено ли устройство к сети/компьютеру (для USB-принтеров — кабель, для сетевых — питание и Wi-Fi/Ethernet).
  • 📄 Установлены ли драйверы (в списке не должно быть значка предупреждения рядом с названием).
  • 🔒 Есть ли у вашей учётной записи права на изменение системных параметров.

Проверьте физическое подключение принтера|Обновите драйверы через официальный сайт производителя|Убедитесь, что принтер включён и не находится в режиме сна|Откройте «Параметры» от имени администратора, если кнопка неактивна-->

В Windows 11 путь немного изменился: Параметры → Bluetooth и устройства → Принтеры и сканеры. Логика действий остаётся той же.

Способ 2: Настройка через «Панель управления» (для старых версий Windows)

Этот метод актуален для Windows 7/8 или если вам удобнее классический интерфейс. Также он полезен в случаях, когда новый интерфейс «Параметров» работает нестабильно.

  1. Откройте Панель управления (можно найти через поиск в меню Пуск).
  2. Перейдите в раздел Устройства и принтеры.
  3. Правой кнопкой мыши кликните на нужный принтер (например, HP LaserJet Pro M404n) и выберите «Использовать по умолчанию».

Зелёная галочка рядом с именем принтера подтвердит, что он теперь основной. Если галочка исчезает после перезагрузки, проблема может быть в конфликте драйверов или повреждённых системных файлах.

⚠️ Внимание: В Windows 7 при смене принтера по умолчанию через «Панель управления» иногда сбрасываются индивидуальные настройки печати (например, ориентация страницы или качество). Перед изменением сохраните профиль настроек в файле .printerExport.

Способ 3: Командная строка и PowerShell для опытных пользователей

Если графический интерфейс недоступен (например, при удалённом администрировании сервера) или нужно автоматизировать процесс, используйте команды.

Для PowerShell (Windows 10/11):

Set-PrintConfiguration -PrinterName "Имя_вашего_принтера" -Priority 1

$printer = Get-Printer -Name "Имя_вашего_принтера"


$printer | Set-Printer -Default

Для классической командной строки (cmd):

rundll32 printui.dll,PrintUIEntry /y /n "Имя_вашего_принтера"

Чтобы узнать точное имя принтера (с учётом пробелов и регистра), выполните:

wmic printer get name
Как проверить, сменился ли принтер по умолчанию через командную строку?

Выполните команду wmic printer get name,default. В колонке Default напротив нужного принтера должно стоять TRUE.

Команда Описание Когда использовать
Set-Printer -Default Назначает принтер основным в PowerShell Для скриптов автоматизации
rundll32 printui.dll Классический метод через cmd Если PowerShell отключён политиками
wmic printer get name,default Показывает текущий принтер по умолчанию Для проверки результата

Способ 4: Изменение принтера по умолчанию в macOS

На компьютерах Apple процесс немного отличается, но не менее интуитивен. Инструкция подходит для macOS Ventura, Monterey и более ранних версий.

  1. Откройте Системные настройки → Принтеры и сканеры.
  2. В левом столбце выберите нужный принтер (например, Canon TS3450).
  3. Кликните правой кнопкой мыши (или удерживая Control) по имени принтера и выберите «Сделать принтером по умолчанию».

Если принтер не отображается в списке:

  • 🔄 Перезагрузите его и компьютер.
  • 📥 Установите драйвер с официального сайта (для моделей Brother или Xerox часто требуется ручная установка).
  • 🌐 Проверьте подключение к сети (для AirPrint или сетевых принтеров).
💡

В macOS можно назначить разные принтеры по умолчанию для чёрно-белой и цветной печати. Для этого в настройках принтера создайте два профиля с разными параметрами.

Способ 5: Настройка через групповую политику (для корпоративных пользователей)

В офисных сетях с доменом Active Directory настройки принтеров часто управляются централизованно. Если вам нужно вручную изменить принтер по умолчанию на рабочем ПК:

  1. Нажмите Win + R, введите gpedit.msc и нажмите Enter.
  2. Перейдите по пути: Конфигурация пользователя → Административные шаблоны → Панель управления → Принтеры.
  3. Найдите политику «Запретить изменение принтера по умолчанию» и убедитесь, что она отключена.

Если политика включена, обратитесь к системному администратору. В некоторых компаниях принтеры по умолчанию назначаются через скрипты входа или политики предпочтительных принтеров, которые перезаписывают локальные настройки.

⚠️ Внимание: В доменных сетях после ручного изменения принтера по умолчанию настройки могут сброситься при следующем входе в систему. Это связано с применением групповой политики (Group Policy Update).

Типичные ошибки и их решения

Даже после выполнения всех шагов принтер может не сохраняться как основной. Рассмотрим распространённые причины и способы их устранения.

Проблема Возможная причина Решение
Кнопка «Назначить по умолчанию» неактивна Отсутствуют права администратора или принтер отключён Запустите «Параметры» от имени админа или проверьте подключение
Настройки сбрасываются после перезагрузки Включена функция «Управление принтерами по умолчанию» Отключите её в Параметры → Устройства → Принтеры и сканеры
Принтер исчезает из списка Конфликт драйверов или повреждённый порт Удалите принтер и добавьте заново, обновите драйверы

Если принтер по умолчанию сбрасывается при подключении к другой сети (например, дома и в офисе), создайте отдельные профили пользователя в Windows или используйте скрипт для автоматического переключения в зависимости от SSID Wi-Fi.

Дополнительные советы по оптимизации печати

Смена принтера по умолчанию — только часть настройки. Чтобы ускорить работу и избежать ошибок:

  • 📁 Сохраняйте профили печати для разных задач (например, «Черновик», «Фото», «Документы»).
  • 🔄 Обновляйте драйверы раз в 3–6 месяцев, особенно для МФУ (Canon imageCLASS, HP OfficeJet Pro).
  • 🌍 Настраивайте сетевые принтеры по статичным IP-адресам, чтобы избежать проблем с подключением.

Для HP и Epson полезно установить фирменное ПО (например, HP Smart или Epson Software Updater), которое предлагает расширенные опции управления, включая приоритеты печати и мониторинг расходников.

💡

Регулярно проверяйте очередь печати (services.msc → Очередь печати), особенно если документы «зависают» или принтер не реагирует на команды.

FAQ: Частые вопросы о смене принтера по умолчанию

Можно ли назначить разные принтеры по умолчанию для разных программ?

Нет, на уровне ОС можно выбрать только один принтер по умолчанию. Однако некоторые программы (например, Adobe Acrobat или Microsoft Word) позволяют сохранять настройки печати для конкретных документов. Также можно использовать специализированное ПО вроде PaperCut для управления печатью в корпоративной среде.

Почему после обновления Windows сбросился принтер по умолчанию?

Это связано с тем, что крупные обновления (например, переход с Windows 10 на 11) сбрасывают некоторые пользовательские настройки. Чтобы избежать проблемы, перед обновлением экспортируйте список принтеров командой printui /L /S и сохраните файл конфигурации.

Как сделать принтер по умолчанию через терминал в Linux?

В большинстве дистрибутивов (Ubuntu, Fedora) используйте команду: 

lpoptions -d Имя_принтера

Чтобы проверить текущий принтер по умолчанию, выполните:

lpstat -d

Можно ли назначить сетевой принтер по умолчанию без подключения к нему?

Технически да, но печать не будет работать, пока принтер не станет доступен в сети. В Windows принтер должен быть в состоянии «Готов», иначе система может проигнорировать команду или вернуть ошибку 0x00000002.

Что делать, если принтер по умолчанию меняется сам после сна/гибернации?

Эта проблема характерна для ноутбуков с режимом энергосбережения USB. Откройте Диспетчер устройств → Контроллеры USB, найдите ваш принтер, перейдите в «Свойства → Управление электропитанием» и снимите галочку с «Разрешить отключение для экономии энергии».
