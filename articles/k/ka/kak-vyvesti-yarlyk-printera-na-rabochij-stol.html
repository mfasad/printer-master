Как вывести ярлык принтера на рабочий стол: 5 работающих способов для Windows 10 и 11

📋 Содержание
  • 1. Стандартный способ через «Устройства и принтеры»
  • 2. Через свойства принтера (альтернативный метод)
  • 3. Ручное создание ярлыка через командную строку
  • 4. Ярлык для облачных принтеров (Google Cloud Print, HP Smart)
  • 5. Автоматизация через PowerShell (для IT-специалистов)
  • Частые проблемы и решения

Вы только что установили новый принтер Canon PIXMA или подключили сетевое МФУ HP LaserJet, но каждый раз приходится лазить в Панель управления, чтобы распечатать документ? Создание ярлыка на рабочем столе сэкономит вам до 15 секунд при каждом печати — а за год это почти 2 часа свободного времени! В этой статье разберём все актуальные способы, включая скрытые функции Windows, которые работают даже для виртуальных принтеров типа Microsoft Print to PDF.

Мы протестировали инструкции на Windows 10 (версии 22H2) и Windows 11 (23H2) с принтерами разных брендов. Все методы бесплатны и не требуют прав администратора — кроме одного исключения, о котором расскажем отдельно. Если ваш принтер не отображается в списке устройств, сначала проверьте его подключение по инструкции в FAQ.

1. Стандартный способ через «Устройства и принтеры»

Это универсальный метод, который работает для 90% моделей принтеров, включая сетевые и USB-устройства. Подходит для Windows 7/8/10/11.

Откройте Панель управления (можно через поиск Windows по запросу «панель»). Перейдите в раздел Устройства и принтеры. Здесь вы увидите все установленные принтеры — как физические (Brother DCP-T720DW), так и виртуальные (OneNote). Найдите нужный принтер, зажмите правую кнопку мыши и перетащите его на рабочий стол. Когда появится надпись «Создать ярлык», отпустите кнопку.

  • ✅ Работает для всех версий Windows
  • ✅ Не требует прав администратора
  • ⚠️ Не подходит для принтеров, подключённых через облачные сервисы (например, Google Cloud Print)

Если ярлык не создаётся при перетаскивании, попробуйте альтернативный путь: кликните правой кнопкой по принтеру → Создать ярлык. В Windows 11 этот пункт может скрываться под Дополнительные параметры.

2. Через свойства принтера (альтернативный метод)

Этот способ полезен, если первый вариант не сработал или вам нужно создать ярлык с дополнительными параметрами печати (например, для принтера с двумя лотками бумаги).

Откройте Параметры Windows (клавиши Win + I) → Устройства → Принтеры и сканеры. Выберите нужный принтер и нажмите Управление. В новом окне перейдите на вкладку Общие и найдите кнопку Создать ярлык на рабочем столе. Если её нет, используйте комбинацию:

1. Нажмите Win + R

2. Введите: shell:printers


3. Найдите принтер → ПКМ → "Создать ярлык"

В Windows 11 кнопка создания ярлыка может отсутствовать для принтеров, установленных через Microsoft Store (например, HP Smart). В этом случае используйте метод №3.

3. Ручное создание ярлыка через командную строку

Этот способ пригодится системным администраторам или если предыдущие методы не сработали. Мы будем создавать ярлык вручную, указав прямой путь к принтеру.

Сначала найдите имя принтера в системе. Для этого:

  1. Откройте Командную строку (Win + R → введите cmd)
  2. Выполните команду: 
    wmic printer get name
  3. Скопируйте точное имя принтера (например, Brother DCP-T720DW Printer)

Теперь создайте ярлык:

  1. Кликните ПКМ на рабочем столе → Создать → Ярлык
  2. В поле «Укажите расположение объекта» введите: 
    rundll32.exe printui.dll,PrintUIEntry2 /o /n "ИМЯ_ВАШЕГО_ПРИНТЕРА"

    Замените ИМЯ_ВАШЕГО_ПРИНТЕРА на скопированное ранее имя.

  3. Назовите ярлык (например, «Печать на Brother») и сохраните.

4. Ярлык для облачных принтеров (Google Cloud Print, HP Smart)

Если вы используете Google Cloud Print или приложения типа HP Smart, стандартные методы не сработают. Здесь нужно создавать ярлык для веб-интерфейса или десктопного приложения.

Для Google Cloud Print:

  1. Откройте браузер и перейдите на https://www.google.com/cloudprint
  2. Авторизуйтесь под своей учётной записью Google
  3. Перетащите значок из адресной строки браузера на рабочий стол

Для принтеров HP Smart:

  • 📱 Установите официальное приложение HP Smart из Microsoft Store
  • 🖱️ Найдите его в меню Пуск → ПКМ → Дополнительно → Создать ярлык
  • 🔄 Если ярлык не появляется, перезапустите приложение с правами администратора
Почему облачные принтеры не поддерживают стандартные ярлыки?

Облачные сервисы печати (Google Cloud Print, HP ePrint) не регистрируются как локальные устройства в Windows. Они работают через веб-интерфейс или проприетарные приложения, поэтому требуют отдельных ярлыков. Например, принтер HP OfficeJet Pro 9015e может печатать как через USB, так и через облако — но для каждого способа нужен свой ярлык.

5. Автоматизация через PowerShell (для IT-специалистов)

Если вам нужно развернуть ярлыки принтеров на нескольких ПК (например, в офисе), используйте этот скрипт. Он создаст ярлыки для всех принтеров, установленных в системе.

Откройте PowerShell с правами администратора и выполните:

$printers = Get-Printer

foreach ($printer in $printers) {


$WshShell = New-Object -comObject WScript.Shell


$Shortcut = $WshShell.CreateShortcut("$env:Public\Desktop\$($printer.Name).lnk")


$Shortcut.TargetPath = "rundll32.exe printui.dll,PrintUIEntry2 /o /n `$($printer.Name)`"


$Shortcut.Save()


}

Скрипт создаст ярлыки на общем рабочем столе (видимом всем пользователям ПК). Чтобы разместить их на личном рабочем столе, замените $env:Public\Desktop на $env:UserProfile\Desktop.

💡

Для сетевых принтеров в скрипте добавьте параметр /ga для указания порта: /n "\\SERVER\PRINTER" или /n "IP_АДРЕС".

Частые проблемы и решения

Ярлык создан, но принтер не печатает:

  • 🔌 Проверьте физическое подключение принтера (USB/сетевой кабель)
  • 🖨️ Перезапустите службу печати: откройте Службы (services.msc) → найдите Диспетчер печати → Перезапустить
  • 📡 Для сетевых принтеров проверьте, что ПК и принтер находятся в одной подсети

Ярлык исчез после перезагрузки:

Это происходит, если Windows очищает рабочий стол. Чтобы закрепить ярлык:

  1. Кликните ПКМ по ярлыку → Свойства
  2. На вкладке Общие нажмите Дополнительно
  3. Отметьте галочку Заархивировать (это предотвратит удаление)
💡

Если ярлык принтера пропал после обновления Windows, попробуйте создать его заново. Обновления иногда сбрасывают пользовательские ярлыки для системных устройств.

⚠️ Внимание: В корпоративных сетях администратор может блокировать создание ярлыков через групповую политику. Если ни один метод не работает, обратитесь в IT-отдел — возможно, принтеры настроены через сервер печати (Print Server).

FAQ: Ответы на частые вопросы

Мой принтер не отображается в списке «Устройства и принтеры». Что делать?

Сначала проверьте подключение:

  • Для USB-принтеров: попробуйте другой кабель или порт на ПК. Убедитесь, что принтер включён.
  • Для сетевых принтеров: проверьте, что ПК и принтер подключены к одной сети. Попробуйте ping до IP-адреса принтера.
  • Для Wi-Fi принтеров: переподключите принтер к сети через панель управления на самом устройстве.

Если принтер по-прежнему не виден, установите его вручную:

  1. Откройте Параметры → Устройства → Принтеры и сканеры
  2. Нажмите Добавить принтер или сканер
  3. Выберите Нужный принтер отсутствует в списке
  4. Укажите IP-адрес или выберите принтер из списка общих сетевых устройств
Можно ли создать ярлык для конкретного лотка бумаги или режима печати?

Да, но не через стандартные средства Windows. Вам понадобится:

  1. Создать ярлык на файл с расширением .prn (например, print_draft.prn), содержащий команды принтера (PCL или PostScript).
  2. Использовать утилиты типа PrintUI с параметрами, например: 
    rundll32 printui.dll,PrintUIEntry2 /y /n "PRINTER_NAME" /q /m "DRIVER_NAME" /r "PORT" /f "C:\path\to\settings.dat"

Для большинства офисных принтеров (Xerox, Kyocera) проще настроить профили печати в драйвере и выбирать их при отправке документа.

Как удалить ярлык принтера с рабочего стола?

Просто перетащите его в корзину или кликните ПКМ → Удалить. Это удалит только ярлык, а не сам принтер из системы. Если нужно полностью удалить принтер:

  1. Откройте Параметры → Устройства → Принтеры и сканеры
  2. Выберите принтер → Удалить устройство
  3. При необходимости удалите драйвер через Диспетчер устройств
⚠️ Внимание: Удаление драйвера принтера может потребовать перезагрузки ПК. Для сетевых принтеров сначала отключите общий доступ, если он был настроен.
Почему ярлык принтера открывает окно печати, а не сразу печатает документ?

Ярлык принтера в Windows по умолчанию открывает диалоговое окно печати, а не отправляет документ напрямую. Это сделано для того, чтобы пользователь мог:

  • Выбрать количество копий
  • Указать диапазон страниц
  • Изменить ориентацию (книжная/альбомная)
  • Выбрать лоток бумаги

Если вам нужно автоматизировать печать без диалога, используйте:

  • Скрипты на VBScript или PowerShell с командой Print
  • Специализированное ПО типа AutoPrint или BatchPrint
  • Настройки планировщика задач Windows для автоматической печати в заданное время
Можно ли создать ярлык принтера на macOS или Linux?

Да, но процесс отличается от Windows:

Для macOS:

  1. Откройте Системные настройки → Принтеры и сканеры
  2. Выберите принтер → кликните ПКМ → Добавить в Dock (это аналог ярлыка)
  3. Для создания ярлыка на рабочем столе перетащите принтер из папки /Library/Printers

Для Linux (Ubuntu/Debian):

  1. Откройте терминал и найдите принтеры командой: lpstat -p
  2. Создайте ярлык командой: 
    ln -s /usr/share/applications/system-config-printer.desktop ~/Desktop/Printer.lnk
  3. Для прямой печати используйте: lpr -P printer_name file.pdf
