Как соединить один принтер с двумя компьютерами: все способы + решение проблем

Организовать печать с двух компьютеров на один принтер можно минимум пятью способами — от простого подключения через USB-хаб до настройки сетевого доступа или использования Google Cloud Print. Выбор метода зависит от модели принтера, операционной системы и даже планировки офиса. Например, для HP LaserJet Pro M404n достаточно включить поддержку сети по Ethernet, а для старого Canon PIXMA MG2540S без Wi-Fi потребуется USB-переключатель или ручная настройка общего доступа в Windows.

В этой статье разберём все актуальные варианты, включая скрытые настройки Windows 11/10 для совместного использования принтеров без лишних программ, а также обойдём типичные ошибки вроде «принтер не виден в сети» или «доступ запрещён». Если вам нужно подключить принтер к двум ноутбукам временно (например, на конференции) или постоянно (домашний офис), здесь найдёте рабочие решения с пошаговыми скриншотами и командами для терминала.

1. Подключение через USB-хаб или переключатель

Самый простой способ для принтеров без сетевых интерфейсов — использовать USB-хаб с кнопкой переключения (например, UGREEN 20212 или Sabrent HB-UMLS>). Такие устройства позволяют физически переключать принтер между двумя ПК нажатием одной кнопки, не отсоединяя кабели. Подходит для моделей вроде Epson L3110 или Brother DCP-T425W, у которых нет Wi-Fi или Ethernet.

Преимущества метода:

  • 🔌 Нет зависимости от сети — работает даже без интернета.
  • 💰 Дешевле сетевого принтера — хаб стоит от 500 рублей.
  • ⚡ Мгновенное переключение без перезагрузки ПК.

Недостатки:

  • ❌ Ручное управление — нужно нажимать кнопку при смене ПК.
  • ❌ Нет одновременной печати — только поочерёдно.
  • ❌ Риск повреждения порта при частых подключениях.

⚠️ Внимание: Некоторые принтеры (например, Canon G3411) могут требовать переустановки драйверов при каждом переключении USB-порта. Перед покупкой хаба проверьте совместимость с вашей моделью на сайте производителя.

2. Настройка общего доступа к принтеру в Windows

Если оба компьютера подключены к одной локальной сети (через роутер или кабель), можно настроить общий доступ к принтеру через основной ПК. Этот метод работает для любых принтеров, даже без сетевых функций, но требует, чтобы «главный» компьютер был включён.

Пошаговая инструкция для Windows 10/11:

  1. Подключите принтер к основному ПК (например, стационарному).
  2. Откройте Параметры → Устройства → Принтеры и сканеры.
  3. Выберите ваш принтер и нажмите Управление → Свойства принтера → Доступ.
  4. Поставьте галочку Общий доступ к данному принтеру и задайте имя (например, HP_OfficeJet_Pro).
  5. На втором ПК откройте Панель управления → Устройства и принтеры → Добавление принтера и выберите Добавить сетевой, беспроводной или Bluetooth-принтер.
  6. В списке найдите принтер с именем, которое задали ранее, и установите драйверы.

Если принтер не отображается:

  • 🔄 Проверьте, что оба ПК в одной рабочей группе (по умолчанию WORKGROUP).
  • 🔥 Отключите брандмауэр временно или добавьте исключение для порта 139 (NetBIOS).
  • 📡 Убедитесь, что сетевое обнаружение включено в Параметры → Сеть и Интернет → Параметры общего доступа.

3. Подключение через Wi-Fi или Ethernet (для сетевых принтеров)

Современные принтеры (HP Neverstop Laser 1000w, Brother HL-L2350DW, Xerox B210D) поддерживают подключение по Wi-Fi или Ethernet. Это позволяет печатать с нескольких устройств одновременно без зависимости от «главного» ПК. Ниже — инструкции для обоих вариантов.

Подключение по Wi-Fi:

  1. На принтере нажмите кнопку Wi-Fi или Сеть (зависит от модели).
  2. Выберите вашу сеть и введите пароль (на дисплее принтера или через WPS).
  3. На ПК добавьте принтер через Параметры → Принтеры и сканеры → Добавление принтера, выбрав вариант Беспроводной принтер.

Подключение по Ethernet:

  1. Соедините принтер с роутером кабелем RJ-45.
  2. Назначьте принтеру статический IP в настройках роутера (например, 192.168.1.100).
  3. На ПК добавьте принтер по IP-адресу: в мастере установки выберите Добавить принтер по TCP/IP-адресу.

Если принтер не подключается к Wi-Fi, попробуйте сбросить его сетевые настройки: нажмите и удерживайте кнопку Wi-Fi (или Reset) 10 секунд, пока индикатор не начнёт мигать. После этого повторите настройку.

4. Использование облачных сервисов (Google Cloud Print, HP Smart)

Облачные сервисы позволяют печатать на принтере с любых устройств, даже вне локальной сети. Например, Google Cloud Print (закрыт в 2020 году, но есть альтернативы) или фирменные решения от производителей:

  • 🖨️ HP Smart — для принтеров HP (поддерживает печать с телефона).
  • 🖨️ Epson Connect — для Epson EcoTank и WorkForce.
  • 🖨️ Canon PRINT — для Canon PIXMA и imageCLASS.

Как настроить HP Smart:

  1. Установите приложение HP Smart на оба ПК (или смартфон).
  2. Создайте аккаунт HP или войдите через Google/Microsoft.
  3. Добавьте принтер в приложение (он должен быть подключён к той же сети).
  4. На втором ПК авторизуйтесь в том же аккаунте — принтер появится автоматически.

Преимущества облачных сервисов:

  • ☁️ Печать из любой точки мира (если принтер включён).
  • 📱 Поддержка мобильных устройств (Android/iOS).
  • 🔄 Автоматическое обновление драйверов.

⚠️ Внимание: Облачные сервисы могут передавать данные о документах на серверы производителя. Для конфиденциальных документов используйте локальную сеть или VPN.

5. Подключение через Raspberry Pi (для продвинутых пользователей)

Если у вас есть Raspberry Pi (или любой другой мини-ПК на Linux), его можно превратить в сервер печати, который будет раздавать доступ к принтеру по сети. Этот метод подходит для старых принтеров без сетевых интерфейсов или для организации печати в небольшом офисе.

Инструкция для Raspberry Pi 4/5:

  1. Подключите принтер к Raspberry Pi по USB.
  2. Установите CUPS (систему печати для Linux): 
    sudo apt update && sudo apt install cups
  3. Добавьте принтер в веб-интерфейсе CUPS (откройте в браузере http://localhost:631).
  4. Настройте общий доступ в файле /etc/cups/cupsd.conf, добавив строки: 
    Allow @LOCAL

    Order allow,deny

  5. На ПК добавьте принтер по IP-адресу Raspberry Pi (например, http://192.168.1.100:631/printers/HP_LaserJet).

Преимущества метода:

  • 🖥️ Работает с любыми принтерами, даже без драйверов для современных ОС.
  • 🔒 Больше контроля над доступом (можно настроить авторизацию).
  • 📊 Ведение логов печати (кто и когда печатал).

Как настроить авторизацию в CUPS?

Чтобы ограничить доступ к принтеру, отредактируйте файл /etc/cups/cupsd.conf:

1. Найдите блок <Location /> и добавьте:

AuthType Basic

AuthClass Group


Require user @SYSTEM

2. Создайте пользователя для печати:

sudo lpadmin -p HP_LaserJet -u allow:username

3. Перезапустите CUPS:

sudo systemctl restart cups

Теперь при добавлении принтера на ПК потребуется ввести логин/пароль.

6. Решение типичных проблем

Даже после правильной настройки могут возникать ошибки. Вот самые распространённые и способы их исправления:

Проблема Возможная причина Решение
Принтер не виден в сети Отключено сетевое обнаружение Включите в Параметры → Сеть → Параметры общего доступа
Ошибка «Доступ запрещён» Недостаточно прав на основном ПК Настройте общий доступ с полными правами для Все
Печать идёт медленно Низкая скорость Wi-Fi или перегрузка роутера Переключите принтер на 5 ГГц или подключите по Ethernet
Драйвер не устанавливается Устаревшее ПО или конфликт версий Скачайте драйвер с сайта производителя, выбрав модель и ОС

Если принтер подключён по USB к хабу, но не определяется:

  • 🔌 Проверьте, что хаб с внешним питанием (пассивные хабы могут не хватать мощности).
  • 🔄 Попробуйте подключить принтер напрямую — если заработает, проблема в хабе.
  • 🛠️ Обновите драйверы чипсета USB на материнской плате.

💡

Перед покупкой USB-хаба для принтера убедитесь, что он поддерживает backward power (обратное питание). Дешёвые хабы могут блокировать сигналы управления, из-за чего принтер не будет корректно определяться.

FAQ: Частые вопросы

Можно ли подключить один принтер к ПК с Windows и Mac одновременно?

Да, но есть нюансы:

  • Для сетевых принтеров (Wi-Fi/Ethernet) достаточно добавить принтер на обоих устройствах через стандартный мастер установки.
  • Для USB-принтеров потребуется либо хаб с переключателем, либо настройка общего доступа через ПК с Windows (на Mac принтер добавляется через Системные настройки → Принтеры и сканеры).
  • На Mac может потребоваться установка драйверов вручную (например, для Brother скачивайте ПО с сайта support.brother.com).

Как подключить принтер к двум компьютерам без роутера?

Есть три варианта:

  1. USB-хаб с переключателем (механическое решение).
  2. Прямое подключение по Ethernet: соедините ПК и принтер кабелем, а на втором ПК настройте общий доступ через первый (нужен кроссовый кабель или адаптер).
  3. Wi-Fi Direct: некоторые принтеры (HP Envy 6055, Canon PIXMA TS8350) поддерживают прямое подключение к ПК без роутера. Настройка ведётся через панель принтера.

Почему принтер печатает документы не с того компьютера, с которого отправляешь?

Эта проблема возникает при настройке очереди печати на основном ПК. Решения:

  • Откройте Службы (services.msc) и перезапустите Диспетчер печати.
  • На основном ПК отключите опцию Оптимизировать работу очереди в свойствах принтера.
  • Если используете Google Cloud Print, проверьте, что в настройках принтера не включён режим Offline.

Можно ли подключить принтер к двум компьютерам через Bluetooth?

Технически да, но на практике это неудобно:

  • Bluetooth-подключение поддерживают редкие модели принтеров (HP Sprocket, некоторые Canon Selphy).
  • Скорость передачи данных низкая — подходит только для фотографий, не для документов.
  • Одновременно подключить два устройства нельзя — нужно каждый раз выполнять сопряжение заново.

Лучше использовать Wi-Fi Direct или USB-хаб.

Как ограничить доступ к принтеру для второго компьютера?

Если принтер подключён через общий доступ Windows:

  1. Откройте Свойства принтера → Безопасность.
  2. Удалите группу Все и добавьте конкретного пользователя.
  3. Настройте права: Печать, Управление документами или Полный доступ.

Для сетевых принтеров (например, Xerox) ограничения настраиваются через веб-интерфейс принтера (введите его IP в браузере).
