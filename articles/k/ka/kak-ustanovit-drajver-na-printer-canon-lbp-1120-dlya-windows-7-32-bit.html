Проблемы с драйвером Canon LBP 1120 на Windows 7? Решение здесь

Лазерный принтер Canon LBP 1120 остаётся популярным выбором для офисов и домашнего использования благодаря своей надёжности и экономичности. Однако владельцы устройств под управлением Windows 7 (32-bit) нередко сталкиваются с трудностями при установке драйверов — система либо не распознаёт принтер, либо выдаёт ошибки совместимости. В этой статье мы разберём три официальных способа установки, включая ручную настройку через диспетчер устройств и решение типичных проблем вроде кода ошибки 0x00000057.

Особенность модели LBP 1120 в том, что Canon больше не поддерживает её на своём сайте, но драйверы всё ещё доступны в архивах и через центр обновления Windows. Мы покажем, где именно искать актуальные файлы (включая альтернативные источники), как правильно подключить принтер через USB или LPT-порт, и что делать, если система упорно не видит устройство. Отдельное внимание уделим совместимости с 32-битной версией Windows 7 — здесь есть нюансы, о которых не пишут в стандартных инструкциях.

Подготовка к установке: что нужно проверить до начала

Прежде чем скачивать драйверы, убедитесь, что ваша система готова к установке. Пропуск этих шагов часто приводит к сбоям вроде "устройство не опознано" или "драйвер не найден".

  • 🔌 Проверьте подключение принтера: используйте оригинальный USB-кабель (не удлинитель!). Для LBP 1120 подходит только USB 2.0 — порты 3.0 (синие) могут не работать.
  • 🖥️ Обновите Windows 7: запустите Центр обновления Windows и установите все критические апдейты. Без пакета KB3033929 принтер может не определиться.
  • 🛠️ Удалите старые драйверы: откройте Панель управления → Программы и компоненты и деинсталлируйте все программы с названием Canon LBP.
  • 🔍 Определите разрядность системы: нажмите Win + Pause, в разделе "Система" проверьте строку "Тип системы" — должно быть указано 32-разрядная.

Если принтер подключён через LPT-порт (параллельный интерфейс), убедитесь, что в BIOS включён режим ECP+EPP. Для этого при загрузке ПК нажмите Del или F2, найдите раздел Integrated Peripherals и активируйте опцию Parallel Port Mode.

Способ 1: Установка драйвера через официальный архив Canon

Несмотря на то, что Canon снял LBP 1120 с поддержки, драйверы для него хранятся в архиве на европейском сайте компании. Следуйте инструкции, чтобы скачать и установить правильную версию:

  1. Перейдите на страницу архива: введите в браузере адрес https://www.canon-europe.com/support/products/imagerunner/laserbase/lbp1120.html (если ссылка не работает, используйте поиск по запросу Canon LBP 1120 driver archive).
  2. Выберите операционную систему: в фильтрах укажите Windows 7 (32-bit). Скачайте файл LBP1120_LBP1120_LBP1120CAPT_R150_V320_W32_RU.exe (версия 3.20 — последняя стабильная для 32-битной системы).
  3. Запустите установщик: откройте скачанный файл от имени администратора (правый клик → Запуск от имени администратора).
  4. Следуйте мастеру установки: на этапе выбора типа подключения укажите USB (даже если используете LPT — драйвер универсальный).

После завершения установки перезагрузите ПК и подключите принтер. Если система запросит дополнительные драйверы, укажите путь к папке C:\Program Files\Canon\LBP1120.

Способ 2: Установка через Центр обновления Windows

Windows 7 содержит базовые драйверы для многих принтеров, включая LBP 1120. Этот метод подходит, если официальный драйвер не устанавливается или вызывают ошибки:

  1. Подключите принтер: соедините Canon LBP 1120 с ПК через USB и включите его. Дождитесь звукового сигнала о обнаружении нового устройства.
  2. Откройте диспетчер устройств: нажмите Win + R, введите devmgmt.msc и нажмите Enter.
  3. Найдите неизвестное устройство: в списке найдите пункт с жёлтым восклицательным знаком (обычно в разделе "Другие устройства" или "Очереди печати").
  4. Обновите драйвер: правый клик по устройству → Обновить драйверы → Автоматический поиск. Система найдёт и установит совместимый драйвер.

Если автоматический поиск не сработал, выберите Выполнить поиск драйверов на этом компьютере → Выбрать драйвер из списка уже установленных. В списке производителей выберите Canon, а в моделях — LBP1120 (может отображаться как Canon LBP1120 CAPT).

Способ 3: Ручная установка драйвера через INF-файл

Этот метод пригодится, если первые два способа не сработали. Мы будем устанавливать драйвер вручную, используя INF-файл из архива Canon. Вот пошаговая инструкция:

  1. Скачайте драйвер-пак: найдите в интернете архив с драйверами для LBP 1120 (например, LBP1120_Driver_V320.zip). Распакуйте его в папку C:\Canon_Driver.
  2. Откройте диспетчер устройств: как в предыдущем способе, найдите неизвестное устройство (принтер с восклицательным знаком).
  3. Укажите путь к INF-файлу: правый клик → Обновить драйверы → Выполнить поиск драйверов на этом компьютере → Выбрать драйвер из списка → Установить с диска. Укажите путь к файлу CNMCA0C.INF в распакованной папке.
  4. Подтвердите установку: игнорируйте предупреждение о несовместимости — драйвер подходит для LBP 1120, несмотря на сообщение.

После установки перезагрузите ПК и проверьте работу принтера. Если в списке портов не отображается USB001 или LPT1, создайте порт вручную:

1. Откройте "Устройства и принтеры".

2. Правый клик по Canon LBP 1120 → "Свойства принтера" → вкладка "Порты".


3. Нажмите "Добавить порт" → "Standard TCP/IP Port" → "Новый порт".


4. Введите IP-адрес (если сетевой) или выберите USB001/LPT1.

⚠️ Внимание: При ручной установке через INF-файл некоторые функции (например, двусторонняя печать) могут не работать. В этом случае дополнительно установите утилиту Canon CAPT с официального сайта.

Решение распространённых ошибок при установке

Даже после успешной установки драйвера пользователи Canon LBP 1120 сталкиваются с ошибками. Рассмотрим самые частые проблемы и способы их устранения:

  • 🔴 Ошибка "Драйвер не найден" (код 28): означает, что Windows не может распознать устройство. Решение: отключите принтер, удалите его из диспетчера устройств, перезагрузите ПК и подключите заново.
  • 🟡 Принтер печатает пустые листы: проверьте уровень тонера и очистите контакты картриджа спиртовой салфеткой. Если проблема остаётся — переустановите драйвер с отметкой "Печать в режиме эмуляции".
  • 🟢 Ошибка "Сбой печати" (код 0x000006d9): сбросьте очередь печати. Для этого откройте Службы (services.msc), найдите Диспетчер печати, остановите службу, удалите файлы из папки C:\Windows\System32\spool\PRINTERS и перезапустите службу.
  • ⚪ Принтер не отвечает (ошибка 0x0000007a): проверьте кабель и порт. Для LPT попробуйте другой разъём или замените шлейф.

Если после установки драйвера принтер определяется, но не печатает, попробуйте следующее:

  1. Откройте Устройства и принтеры, правый клик по LBP 1120 → Свойства принтера → вкладка Общие → Печать пробной страницы.
  2. Если пробная страница не печатается, перейдите на вкладку Порты и убедитесь, что выбран правильный порт (USB001 или LPT1).
  3. На вкладке Дополнительно проверьте, что в поле "Драйвер" указано Canon LBP1120 CAPT, а не универсальный драйвер Windows.
Что делать, если принтер определяется как "Неопознанное устройство"?

Это означает, что Windows не может подобрать драйвер автоматически. Решение:

1. Скачайте драйвер с сайта Canon (см. Способ 1).

2. Откройте диспетчер устройств, найдите "Неопознанное устройство".

3. Правый клик → "Обновить драйверы" → "Выполнить поиск на этом компьютере".

4. Укажите папку с распакованным драйвером (например, C:\Canon_Driver).

5. После установки перезагрузите ПК.

Альтернативные источники драйверов: где искать, если официальный сайт не помогает

Если драйвер с сайта Canon не устанавливается или вызывает ошибки, воспользуйтесь проверенными альтернативными источниками. Важно скачивать файлы только с надёжных ресурсов, чтобы избежать вирусов.

  • 🌍 Сайт DriverPack Solution: программа автоматически найдёт и установит драйвер, но будьте осторожны — отключите установку дополнительного ПО в процессе.
  • 📦 Архив драйверов на Softpedia: ищите по запросу Canon LBP 1120 driver Windows 7 32-bit. Скачивайте только файлы с пометкой WHQL (сертифицированные Microsoft).
  • 💾 Диск с драйверами от принтера: если сохранился компакт-диск из комплекта, используйте его. Для этого вставьте диск в привод и запустите Setup.exe.
  • 🔄 Драйверы от аналогичных моделей: в крайнем случае подойдёт драйвер от Canon LBP 1210 (совместим на 90%). Скачайте его с официального сайта и установите вручную через INF-файл.

При скачивании с сторонних сайтов всегда проверяйте файлы антивирусом. Мы рекомендуем использовать VirusTotal для проверки скачанных архивов. Также обращайте внимание на дату выпуска драйвера — актуальные версии для LBP 1120 не новее 2015 года.

⚠️ Внимание: Некоторые сайты предлагают "универсальные драйверы" для всех моделей Canon. Эти файлы часто содержат вредоносное ПО или не обеспечивают полную функциональность. Используйте только драйверы, предназначенные конкретно для LBP 1120.
💡

Если принтер подключён через USB, но Windows его не видит, попробуйте подключить к другому порту или использовать USB-хаб с внешним питанием. Проблема может быть в нехватке энергии для устройства.

FAQ: Частые вопросы по установке драйвера Canon LBP 1120

Можно ли установить драйвер для Canon LBP 1120 на Windows 10/11?

Да, но с оговорками. Официально Canon не поддерживает LBP 1120 на Windows 10/11, но драйвер от Windows 7 (32-bit) часто работает в режиме совместимости. Для этого:

  1. Скачайте драйвер для Windows 7 (32-bit) с сайта Canon.
  2. Правый клик по установщику → Свойства → вкладка Совместимость.
  3. Отметьте Запустить программу в режиме совместимости с Windows 7.
  4. Запустите установщик от имени администратора.

Если принтер не определяется, попробуйте установить его через Добавление принтера вручную, выбрав модель LBP 1120 из списка.

Почему принтер Canon LBP 1120 печатает с полосами или бледным текстом?

Эта проблема связана с износом фотобарабана или картриджа. Попробуйте следующие шаги:

  • Извлеките картридж и аккуратно потрясите его из стороны в сторону (это распределит тонер равномерно).
  • Очистите контакты картриджа и принтера сухой салфеткой.
  • В настройках драйвера (Свойства принтера → Качество печати) выберите режим Тонер: Темнее.
  • Если полосы остаются — требуется замена фотобарабана или картриджа.
Как сбросить счётчик тонера на Canon LBP 1120?

Сброс счётчика тонера возможен только через сервисное меню принтера. Для этого:

  1. Выключите принтер.
  2. Зажмите кнопку Resume (с треугольником) и удерживайте её.
  3. Включите принтер, продолжая удерживать Resume.
  4. Когда загорится зелёный индикатор, отпустите кнопку и нажмите её 2 раза подряд.
  5. Счётчик тонера будет сброшен, но помните: это не добавит тонер в картридж, а только обнулит показания!

⚠️ Частый сброс счётчика без замены картриджа может привести к поломке принтера.

Где найти оригинальный диск с драйверами для Canon LBP 1120?

Оригинальный диск из комплекта Canon LBP 1120 давно не поставляется, но его содержимое можно найти в интернете. Попробуйте следующие варианты:

  • Поиск по запросу Canon LBP 1120 original driver CD ISO на торрент-трекерах или файлообменниках.
  • Запрос на тематических форумах (например, 4PDA или IXBT).
  • Покупка б/у диска на Avito или Юле (уточните у продавца, что диск именно для LBP 1120).

Если диск найдётся, смонтируйте его образ через Daemon Tools или запишите на флешку с помощью Rufus.

Можно ли подключить Canon LBP 1120 по сети (Wi-Fi или Ethernet)?

Нет, Canon LBP 1120 не поддерживает сетевое подключение. Эта модель оснащена только интерфейсами USB 2.0 и LPT (параллельный порт). Для совместного использования по сети вам потребуется:

  • Подключить принтер к одному из ПК и расшарить его (открыть общий доступ).
  • Использовать специализированный принт-сервер (например, TP-Link TL-PS110U), подключённый к принтеру через USB.

Для настройки общего доступа:

1. Откройте "Устройства и принтеры".

2. Правый клик по Canon LBP 1120 → "Свойства принтера" → вкладка "Доступ".


3. Отметьте "Общий доступ к данному принтеру" и задайте имя.


4. На других ПК добавьте принтер через "Добавление сетевого принтера".
