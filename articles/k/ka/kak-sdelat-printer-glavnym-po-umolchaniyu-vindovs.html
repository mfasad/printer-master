Как сделать принтер основным по умолчанию в Windows: 5 проверенных способов

Работа с документами в офисе или дома требует, чтобы принтер по умолчанию был настроен корректно. Но Windows не всегда сохраняет выбранное устройство: после перезагрузки, обновления системы или подключения нового оборудования основной принтер может сброситься на Microsoft Print to PDF или другой виртуальный драйвер. Это приводит к пустой трате времени — документы уходят не туда, а пользователю приходится вручную выбирать устройство при каждой печати.

В этой статье разберём все актуальные способы сделать принтер главным по умолчанию — от стандартных настроек Windows до командной строки и редактирования реестра. Особое внимание уделим проблемам, когда система автоматически сбрасывает принтер по умолчанию после сна или перезагрузки, и расскажем, как это исправить раз и навсегда.

Если вы используете принтер в связке со сканером штрих-кода или МФУ (многофункциональным устройством), правильная настройка по умолчанию сэкономит время при печати этикеток, накладных или чеков. Например, в торговле или на складе, где скорость обработки документов критична, каждый лишний клик может задерживать работу.

Почему Windows сбрасывает принтер по умолчанию?

Проблема нестабильного сохранения основного принтера знакома многим пользователям. Причины могут быть разными — от багов системы до конфликтов драйверов. Вот самые распространённые сценарии:

  • 🔄 Автоматическое управление Windows. Начиная с Windows 10 (версия 1803), система по умолчанию использует функцию Let Windows manage my default printer, которая выбирает последний использованный принтер. Это удобно для пользователей с несколькими устройствами, но мешает тем, кто работает с одним принтером.
  • 🖨️ Конфликты драйверов. Если принтер подключён по USB и одновременно установлен как сетевой, Windows может "путаться" между двумя экземплярами одного устройства.
  • 🔌 Проблемы с подключением. Принтеры, подключённые по Wi-Fi или через сетевой кабель, иногда "исчезают" из системы после сна или перезагрузки маршрутизатора.
  • 📥 Обновления системы. Крупные апдейты Windows (например, переход с Windows 10 на Windows 11) могут сбрасывать настройки печати.

Если ваш принтер регулярно "слетает" с позиции основного, сначала проверьте, не включена ли функция автоматического управления. Для этого перейдите в Параметры → Устройства → Принтеры и сканеры и отключите опцию Разрешить Windows управлять принтером по умолчанию.

⚠️ Внимание: На некоторых корпоративных ПК настройки принтера могут блокироваться групповой политикой. Если пункты меню недоступны (затенены серым), обратитесь к системному администратору.

Способ 1: Настройка через "Параметры Windows" (самый простой)

Это базовый метод, который работает в Windows 10 и Windows 11. Подходит для большинства пользователей, если принтер уже установлен в системе и не требует дополнительных драйверов.

  1. Откройте меню Пуск и выберите Параметры (значок шестерёнки).
  2. Перейдите в раздел Устройства → Принтеры и сканеры.
  3. В списке устройств найдите нужный принтер и кликните по нему левой кнопкой мыши.
  4. Нажмите кнопку Управление.
  5. В новом окне выберите Установить по умолчанию.

Если кнопка Установить по умолчанию неактивна, это означает, что:

  • 🔹 Принтер уже является основным.
  • 🔹 Устройство отключено или недоступно (проверьте подключение по USB/сети).
  • 🔹 Драйвер принтера установлен некорректно (требуется переустановка).

Способ 2: Через "Панель управления" (для старых версий Windows)

Если вы работаете на Windows 7/8.1 или предпочитаете классический интерфейс, можно использовать Панель управления. Этот метод также полезен, если новый интерфейс "Параметров" работает нестабильно.

  1. Нажмите Win + R, введите control printers и нажмите Enter.
  2. В открывшемся окне найдите нужный принтер, кликните по нему правой кнопкой мыши.
  3. В контекстном меню выберите Использовать по умолчанию.
  4. Закройте окно и перезагрузите ПК для применения изменений.

Если принтер отсутствует в списке, проверьте его подключение или установите драйверы заново. Для сетевых принтеров может потребоваться ввод IP-адреса вручную (об этом — в следующем разделе).

⚠️ Внимание: В Windows 11 классическая "Панель управления" постепенно удаляется. Если команда control printers не работает, используйте способ 1 или 3.

Способ 3: Настройка сетевого принтера по умолчанию

Сетевые принтеры (подключённые по Wi-Fi, Ethernet или через print-сервер) часто требуют дополнительных действий. Если принтер не отображается в списке устройств, его нужно добавить вручную.

Шаг 1: Добавление сетевого принтера

  1. Перейдите в Параметры → Bluetooth и устройства → Принтеры и сканеры.
  2. Нажмите Добавить устройство → Добавить вручную.
  3. Выберите Добавить принтер по TCP/IP-адресу или имени узла.
  4. Введите IP-адрес принтера (например, 192.168.1.100) и нажмите Далее.
  5. Если система запросит драйвер, выберите модель из списка или установите с официального сайта производителя.

Шаг 2: Установка по умолчанию

После добавления принтера установите его основным через Параметры (способ 1) или Панель управления (способ 2).

Для офисных сетей с Active Directory принтеры могут развёртываться через групповую политику. В этом случае настройки по умолчанию могут блокироваться администратором.

Как найти IP-адрес сетевого принтера?

На большинстве принтеров IP-адрес можно узнать, распечатав тестовый лист (нажмите и удерживайте кнопку питания или специальную кнопку "Info"). Также адрес отображается в веб-интерфейсе маршрутизатора (раздел "Подключённые устройства").

Способ 4: Через командную строку (для опытных пользователей)

Если графический интерфейс недоступен (например, на серверных версиях Windows) или нужно автоматизировать процесс, можно использовать командную строку или PowerShell.

Вариант 1: Command Prompt (CMD)

Откройте командную строку от имени администратора и выполните: 

wmic printer where name="ИМЯ_ПРИНТЕРА" call setdefaultprinter

Замените ИМЯ_ПРИНТЕРА на точное название вашего устройства (можно скопировать из списка в Принтеры и сканеры).

Чтобы увидеть список всех принтеров, используйте команду: 

wmic printer get name

Вариант 2: PowerShell

Запустите PowerShell с правами администратора и выполните: 

$printer = Get-Printer -Name "ИМЯ_ПРИНТЕРА"

$printer | Set-Printer -Default

Эти команды полезны для скриптов автоматической настройки или удалённого управления принтерами на нескольких ПК.

⚠️ Внимание: При использовании командной строки учитывайте регистр символов в имени принтера. Если в названии есть пробелы, заключите его в кавычки.
💡

Создайте ярлык на рабочем столе с командой для быстрого сброса принтера по умолчанию. Для этого создайте новый ярлык, в поле "Расположение" введите cmd /c wmic printer where name="ИМЯ_ПРИНТЕРА" call setdefaultprinter, а затем измените значок на любой удобный.

Способ 5: Редактирование реестра (если ничего не помогает)

Если принтер упорно сбрасывается после перезагрузки, проблема может крыться в реестре Windows. Этот метод требует осторожности — неправильные изменения могут нарушить работу системы.

  1. Нажмите Win + R, введите regedit и нажмите Enter.
  2. Перейдите по пути: 
    HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\Windows
  3. Найдите параметр LegacyDefaultPrinterMode. Если его нет — создайте его:
    • Кликните правой кнопкой → Создать → Параметр DWORD (32 бита).
    • Назовите его LegacyDefaultPrinterMode.
    • Установите значение 1.
  • Также проверьте параметр Device в этом же разделе — он должен содержать имя вашего принтера (например, \\server\printer или HP LaserJet M404n).
  • Перезагрузите ПК.

    • Этот метод отключает функцию автоматического управления принтерами, которая появилась в Windows 10. После изменений принтер по умолчанию будет сохраняться даже после обновлений.

    Параметр реестра Значение Описание
    LegacyDefaultPrinterMode 1 Отключает автоматическое управление принтерами
    Device \\server\printer или Имя_принтера Указывает принтер по умолчанию
    UserSelectedDefault Имя_принтера Подтверждает ручной выбор пользователя
    ⚠️ Внимание: Перед редактированием реестра создайте точку восстановления системы (Панель управления → Восстановление → Настройка восстановления системы).
    💡

    Редактирование реестра — крайний метод. Используйте его только если другие способы не сработали, и вы уверены в своих действиях.

    Решение проблемы с автоматическим сбросом принтера

    Если Windows упорно сбрасывает принтер по умолчанию, даже после ручной настройки, проверьте следующие моменты:

    • 🔧 Отключите автоматическое управление: как упоминалось ранее, в Параметры → Принтеры и сканеры должна быть отключена опция Разрешить Windows управлять принтером по умолчанию.
    • 🔄 Обновите драйверы: заходите на сайт производителя принтера (например, HP, Canon, Brother) и скачивайте самую свежую версию ПО.
    • 📡 Проверьте сетевое подключение: для Wi-Fi принтеров установите статический IP-адрес в настройках маршрутизатора, чтобы избежать конфликтов DHCP.
    • 🛠️ Используйте утилиты производителя: многие бренды (например, Epson или Xerox) предлагают фирменное ПО для управления принтерами.

    Для корпоративных пользователей: если принтер развёрнут через print-сервер, настройки по умолчанию могут управляться администратором. В этом случае изменения на локальном ПК не сохранятся.

    Как обновить драйвер принтера?

    1. Откройте "Диспетчер устройств" (Win + X → Диспетчер устройств).

    2. Найдите раздел "Очереди печати", раскройте его.

    3. Кликните правой кнопкой по принтеру → "Обновить драйвер" → "Автоматический поиск".

    Если Windows не находит обновлений, скачайте драйвер с сайта производителя и выберите "Выполнить поиск драйверов на этом компьютере".

    Частые ошибки и их решение

    При настройке принтера по умолчанию пользователи сталкиваются с типичными проблемами. Разберём самые распространённые:

    Ошибка: "Невозможно установить принтер по умолчанию"

    Причины и решения:

    • 🔌 Принтер отключён: проверьте питание, кабели USB/сети, включите устройство.
    • 🖥️ Драйвер не установлен: в "Диспетчере устройств" рядом с принтером может гореть восклицательный знак. Переустановите драйвер.
    • 🔒 Ограничения групповой политики: в корпоративных сетях настройки могут блокироваться. Обратитесь к IT-отделу.

    Ошибка: Принтер исчезает из списка после сна/перезагрузки

    Это характерно для сетевых принтеров. Решения:

    • 📡 Настройте статический IP для принтера в маршрутизаторе.
    • 🔄 Включите функцию Wake on LAN в настройках принтера (если поддерживается).
    • 🖨️ Подключите принтер по USB вместо Wi-Fi (если возможно).

    Ошибка: Печать идёт на другой принтер, despite настройкам

    Проверьте:

    • 📄 В программе (например, Word или Excel) может быть выбран другой принтер. Перед печатью всегда уточняйте устройство в диалоговом окне.
    • 🔧 В некоторых приложениях (например, 1С или AutoCAD) настройки принтера сохраняются отдельно. Настройте печать непосредственно в программе.

    FAQ: Частые вопросы

    Можно ли сделать принтер по умолчанию для конкретной программы?

    Да, некоторые приложения (например, Adobe Acrobat или Microsoft Office) позволяют сохранять настройки принтера отдельно. Для этого:

    1. Откройте документ и нажмите Ctrl + P.
    2. Выберите нужный принтер в выпадающем меню.
    3. Нажмите Свойства принтера и сохраните настройки.

    В 1С или AutoCAD настройки печати могут храниться в шаблонах документов.

    Почему после обновления Windows принтер сбрасывается?

    Крупные обновления (например, Windows 10 22H2 или переход на Windows 11) могут сбрасывать пользовательские настройки, включая принтер по умолчанию. Это связано с:

    • Изменениями в службе печати (Print Spooler).
    • Обновлением драйверов устройств.
    • Сбросом пользовательского профиля (редко).

    Решение: после обновления заново настройте принтер по умолчанию и отключите автоматическое управление (способ 1).

    Как сделать принтер по умолчанию для всех пользователей на одном ПК?

    Для этого нужно:

    1. Открыть Панель управления → Администрирование → Управление печатью.
    2. Найти нужный принтер, кликнуть правой кнопкой и выбрать Установить по умолчанию.
    3. В свойствах принтера перейти на вкладку Безопасность и настроить права для группы Все или Пользователи.

    В Windows 10/11 Pro и корпоративных версиях это можно сделать через групповую политику (gpedit.msc).

    Мой принтер HP LaserJet не сохраняется как основной. Что делать?

    Для принтеров HP (особенно серий LaserJet, OfficeJet, DeskJet) характерны проблемы с драйверами. Попробуйте:

    1. Удалите текущий драйвер через Панель управления → Программы и компоненты.
    2. Скачайте полный пакет ПО с сайта HP Support (выберите точную модель).
    3. Установите драйвер в режиме Администратор.
    4. После установки перезагрузите ПК и настройте принтер по умолчанию.

    Если проблема остаётся, отключите службу HP Network Devices Support в Службах Windows (services.msc).

    Можно ли настроить автоматический выбор принтера в зависимости от сети?

    Да, но для этого потребуется:

    • 🖥️ Скрипт на PowerShell, который будет проверять текущую сеть и устанавливать принтер по умолчанию.
    • 📡 Групповая политика (в корпоративных версиях Windows).
    • 🛠️ Стороннее ПО, например, Default Printer Switcher или PrinterLogic.

    Пример скрипта для PowerShell:

    # Проверяем текущую сеть

    $currentNetwork = (Get-NetIPConfiguration | Where-Object {$_.IPv4DefaultGateway -ne $null}).InterfaceAlias
    


    Устанавливаем принтер по умолчанию в зависимости от сети
    

    if ($currentNetwork -like "Wi-Fi") {
    

    $printer = Get-Printer -Name "HP_Office_WiFi"
    

    $printer | Set-Printer -Default
    

    }
    

    else {
    

    $printer = Get-Printer -Name "Xerox_Office_LAN"
    

    $printer | Set-Printer -Default
    

    }

    Сохраните скрипт как .ps1 и добавьте его в Планировщик заданий для выполнения при входе в систему.
