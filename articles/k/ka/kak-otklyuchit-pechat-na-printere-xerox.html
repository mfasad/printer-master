Как полностью отключить печать на принтере Xerox: инструкции для всех моделей

Отключение функции печати на принтерах Xerox может понадобиться в офисах для ограничения доступа, экономии расходных материалов или временной блокировки устройства. Процедура варьируется в зависимости от модели (WorkCentre 78xx, Phaser 6510, VersaLink C405) и версии прошивки, но основные методы универсальны. В этой статье — проверенные способы с пояснениями для администраторов и обычных пользователей.

Важно понимать разницу между временной блокировкой (например, через пароль) и полным отключением (изменение конфигурации). Первый вариант подходит для ограничения доступа, второй — для сервисного обслуживания или перепрофилирования устройства. Если принтер подключён к сети, изменения могут потребовать прав администратора.

1. Отключение печати через панель управления принтера

Самый быстрый способ — использовать встроенное меню. Подходит для большинства моделей Xerox с сенсорным экраном или кнопочным управлением. Инструкция актуальна для WorkCentre 33xx/58xx/78xx и VersaLink:

  1. Авторизуйтесь как администратор. Нажмите Machine Status → Tools → Admin Login. По умолчанию пароль часто пустой или 1111/admin.
  2. Перейдите в Settings → System Settings → Print Settings.
  3. Найдите опцию Printing Enabled или Allow Printing и установите значение Off.
  4. Сохраните изменения и перезагрузите принтер (кнопка Power удерживается 5 секунд).

На моделях без сенсорного экрана (например, Phaser 3260) путь может отличаться: Menu → System Setup → Print Options → Disable Printing.

2. Блокировка печати через веб-интерфейс (CentreWare)

Удалённое управление доступно для сетевых принтеров Xerox через встроенный веб-сервер CentreWare Internet Services (CWIS). Этот метод подходит для IT-администраторов и позволяет гибко настраивать права доступа:

Шаги для входа в веб-интерфейс:

  1. Узнайте IP-адрес принтера (распечатайте Configuration Page или проверьте в роутере).
  2. Введите IP в адресной строке браузера (например, http://192.168.1.100).
  3. Авторизуйтесь с правами администратора (логины по умолчанию: admin/1111 или sysadm/sysadm).

Далее перейдите в раздел Properties → Services → Printing и:

  • 🔧 Установите флажок Disable Printing для полного отключения.
  • 🔒 Или настройте Access Control, чтобы ограничить печать по IP/MAC-адресам.
  • 📝 Для временной блокировки используйте Schedule (например, отключить печать с 18:00 до 8:00).

Если не удаётся войти в CWIS, сбросьте сетевые настройки принтера удержанием кнопки Go (или Tick) 10 секунд. Это вернёт IP к заводскому 192.168.1.1.

3. Программное отключение через Xerox Print Drivers

Для принтеров, подключённых к ПК, печать можно заблокировать на уровне драйвера. Этот метод не отключает само устройство, но предотвращает отправку заданий с конкретного компьютера:

Инструкция для Windows:

  1. Откройте Панель управления → Устройства и принтеры.
  2. Правой кнопкой по принтеру Xerox → Настройки принтера.
  3. Перейдите на вкладку Порты и снимите галочку с Включить двунаправленную поддержку.
  4. На вкладке Дополнительно выберите Печать напрямую на принтер (это отключит обработку заданий).

Для macOS:

  • 🖥️ Откройте Системные настройки → Принтеры и сканеры.
  • 🔄 Выберите принтер Xerox → Параметры и расходные материалы → Драйвер.
  • 🚫 Установите Печать: Отключено (если опция доступна).
⚠️ Внимание: На некоторых версиях драйверов Xerox Global Print Driver опция полного отключения печати отсутствует. В этом случае используйте групповую политику Windows (gpedit.msc) для блокировки доступа к принтеру.

4. Физические способы блокировки печати

Если программные методы недоступны (например, нет прав администратора), можно применить физические ограничения. Они менее элегантны, но эффективны:

  • 🔌 Отключение кабелей: USB/ETHернет-кабель от принтера к ПК/роутеру. Для сетевых моделей — отключите патч-корд от порта LAN.
  • 🔐 Блокировка лотка: На некоторых моделях (WorkCentre 7970) есть механический замок на лотке для бумаги.
  • 🔧 Извлечение тонера: Без картриджа принтер не сможет печатать (но это может привести к ошибкам 10.000.48).
  • 🛡️ Использование Kensingto-замка: Подходит для офисных МФУ с соответствующим разъёмом.

Для моделей с жёстким диском (например, Xerox AltaLink B81xx) физическое отключение питания может привести к потере данных в очереди печати. В этом случае лучше использовать программные методы.

Что делать если принтер выдаёт ошибку после извлечения тонера?

После извлечения картриджа принтеры Xerox могут показывать ошибки 092-67x или 10.000.48. Для сброса:

1. Включите принтер удерживая кнопку Cancel 10 секунд.

2. В меню сервиса выберите Clear All Errors.

3. Установите картридж обратно или подтвердите работу без него (если поддерживается).

5. Отключение печати через сервер печати (для сетевых принтеров)

В корпоративных сетях печать часто управляется через сервер (Windows Print Server, CUPS на Linux). Блокировка на этом уровне позволит отключить принтер для всех пользователей одновременно:

Для Windows Server:

1. Откройте "Диспетчер сервера" → "Службы печати".

2. Найдите принтер Xerox в списке, правой кнопкой → "Приостановить печать".


3. Для полного отключения: "Свойства принтера" → вкладка "Безопасность" → отмените все разрешения.

Для Linux (CUPS):

sudo lpadmin -p Xerox_Printer_Name -E -o printer-is-shared=false

sudo cupsdisable Xerox_Printer_Name

Этот метод не влияет на прямой IP-доступ к принтеру, но блокирует печать через сервер. Для полной изоляции устройства измените его IP-адрес в настройках сети или отключите порт на коммутаторе.

💡

Отключение на сервере печати — самый надёжный способ для офисов, так как не зависит от настроек самого принтера и блокирует доступ всем пользователям сети.

6. Сброс принтера к заводским настройкам (крайний метод)

Если другие способы не сработали или принтер заражён вредоносным ПО, изменяющим настройки, поможет полный сброс. Это удалит все пользовательские конфигурации, включая сетевые параметры:

Инструкция для большинства моделей:

  1. Выключите принтер.
  2. Удерживайте кнопку 0 (ноль) на панели и включите питание.
  3. Держите 0 до появления меню Service Mode (обычно 10–15 секунд).
  4. Выберите NVM Initialization или Factory Defaults.
  5. Подтвердите сброс (может потребоваться ввод кода 6789).

После сброса принтер вернётся к состоянию "из коробки". Для моделей с жёстким диском (AltaLink, WorkCentre 7xxx) это также удалит все сохранённые документы и очереди печати.

⚠️ Внимание: На принтерах Xerox с прошивкой ConnectKey (2018 года и новее) сброс может потребовать повторной активации лицензий на функции (например, Secure Print или Mobile Print). Коды лицензий обычно указаны на наклейке сзади устройства.

FAQ: Ответы на популярные вопросы

Можно ли отключить печать только для определённых пользователей?

Да, через веб-интерфейс CentreWare (раздел Access Control) или групповую политику Windows. На принтерах с ConnectKey поддерживается интеграция с Active Directory для гибких разрешений.

Почему после отключения печати принтер всё равно принимает задания?

Скорее всего, блокировка применена только к локальным задачам. Проверьте:

  • Настройки Print Services в CWIS (должно быть Disable All Jobs).
  • Очередь печати на сервере (возможно, задания накопились до блокировки).

Для полной остановки отключите порт 9100 на роутере.

Как вернуть печать после отключения?

Включите функцию обратно в меню принтера или CWIS. Если использовался сброс к заводским настройкам, потребуется повторная конфигурация сети и установка драйверов на ПК. Для моделей с жёстким диском может понадобиться восстановление данных из резервной копии.

Будет ли принтер потреблять меньше электроэнергии после отключения печати?

Нет, блокировка печати не влияет на энергопотребление. Для экономии используйте режим Sleep Mode (настраивается в Power Saver Settings) или физически отключайте питание.

Можно ли отключить печать удалённо, если принтер не в локальной сети?

Да, если принтер подключён к облаку Xerox через ConnectKey Apps. Используйте портал Xerox App Gallery для управления. Альтернатива — VPN-подключение к офисной сети и доступ через CWIS.
