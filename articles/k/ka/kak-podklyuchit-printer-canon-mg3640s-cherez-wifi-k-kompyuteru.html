Как подключить принтер Canon MG3640S к компьютеру через Wi-Fi: полное руководство с картинками

Беспроводное подключение принтера Canon PIXMA MG3640S к компьютеру экономит место на рабочем столе и позволяет печатать с нескольких устройств одновременно. Однако многие пользователи сталкиваются с трудностями при первой настройке: принтер не находит сеть, драйвера не устанавливаются, или соединение постоянно разрывается. В этой статье мы разберём все этапы подключения — от подготовки оборудования до устранения типичных ошибок.

Особенность модели MG3640S заключается в поддержке стандарта Wi-Fi Direct, что позволяет подключаться даже без роутера. Но для стабильной работы с ПК лучше использовать классический вариант через домашнюю сеть. Мы рассмотрим оба способа, а также расскажем, как проверить совместимость вашего компьютера с принтером и где скачать официальные драйвера.

Если вы ранее не настраивали сетевые устройства, не переживайте: инструкция адаптирована для новичков. Все шаги сопровождаются скриншотами интерфейса Canon и пояснениями технических терминов. А для опытных пользователей мы добавили раздел с альтернативными методами подключения и расширенными настройками сети.

1. Подготовка принтера и компьютера к подключению

Перед настройкой Wi-Fi убедитесь, что ваш Canon MG3640S и компьютер соответствуют минимальным требованиям. Принтер должен быть распакован, заправлен чернилами и подключён к электросети. На передней панели не должно гореть красных индикаторов ошибок (например, Е03 — нет бумаги, Е16 — проблемы с картриджем).

С компьютера проверьте:

  • 🖥️ Операционная система: Windows 7/8/10/11 или macOS 10.12+. Для Linux потребуются дополнительные драйвера (раздел 5).
  • 📶 Wi-Fi адаптер: должен поддерживать стандарты 802.11b/g/n (все современные ноутбуки и ПК подходят).
  • 🔌 Свободный USB-порт: временно понадобится для установки драйверов, если принтер не определяется по воздуху.

Важно: если ваш роутер работает в режиме 5 ГГц, принтер MG3640S его не увидит — модель поддерживает только 2.4 ГГц. При необходимости переключите роутер в смешанный режим или создайте отдельную сеть для устройств на 2.4 ГГц.

⚠️ Внимание: Некоторые провайдеры по умолчанию отключают broadcast SSID (скрывают имя сети). Если ваш Wi-Fi не отображается в списке доступных на принтере, временно включите видимость сети в настройках роутера.

Также подготовьте:

  • 📄 Документы: серийный номер принтера (на наклейке сзади) и пароль от Wi-Fi.
  • 💿 Диск с ПО: если он шёл в комплекте, можно использовать его. Но мы рекомендуем скачать свежую версию с сайта Canon.
  • 📱 Смартфон: понадобится для альтернативного метода настройки через приложение Canon PRINT (раздел 4).

2. Установка драйверов и программного обеспечения

Без правильных драйверов ваш компьютер просто не распознает принтер, даже если тот подключён к той же сети. Для Canon MG3640S необходимо установить два компонента:

  1. Драйвер принтера — обеспечивает печать.
  2. Утилиту IJ Network Tool — помогает настроить сетевое подключение.

Скачайте их с официального сайта Canon (раздел "Поддержка" → введите модель MG3640S → выберите вашу ОС). Не используйте драйвера с сторонних сайтов — они могут содержать вирусы или быть устаревшими. Объём загрузки: ~150 МБ для полного пакета.

Процесс установки:

  1. Запустите скачанный файл (например, mg3600series-full.exe).
  2. Выберите "Рекомендуемая установка" — это установит все необходимые компоненты.
  3. На этапе "Тип подключения" выберите "Беспроводная сеть".
  4. Дождитесь завершения — компьютер перезагружать не нужно.
⚠️ Внимание: Если установщик предлагает подключить принтер по USB на этом этапе — откажитесь. Мы настроим соединение по Wi-Fi вручную.

После установки в меню "Пуск" появятся две важные программы:

  • Canon IJ Network Tool — для настройки сети.
  • Canon MG3600 series Printer Driver — для управления печатью.

3. Подключение принтера к Wi-Fi сети

Теперь перейдём к самому важному этапу — подключению принтера к вашей беспроводной сети. Есть два основных способа: через панель управления принтера и с помощью компьютера. Мы разберём оба.

Способ 1: Настройка через панель принтера

  1. Включите принтер кнопкой ON/OFF (спереди слева).
  2. Нажмите и удерживайте кнопку Wi-Fi (с иконкой антенны) до тех пор, пока индикатор не начнёт мигать.
  3. На экране принтера выберите "Настройка LAN" → "Беспроводная связь" → "Стандартная настройка".
  4. Выберите вашу Wi-Fi сеть из списка. Если её нет — нажмите "Другое" и введите SSID вручную.
  5. Введите пароль от Wi-Fi (регистр важен!). Используйте кнопки со стрелками для навигации.
  6. Подтвердите настройки. Индикатор Wi-Fi должен гореть зелёным — это значит, что подключение успешно.

Способ 2: Настройка через компьютер (с помощью IJ Network Tool)

  1. Запустите программу Canon IJ Network Tool.
  2. В окне программы нажмите "Настройка" → "Подключение по беспроводной сети".
  3. Выберите "Подключение через кабель USB" (временное подключение для передачи настроек).
  4. Следуйте инструкциям мастера, выбрав вашу Wi-Fi сеть и введя пароль.
  5. После завершения отключите USB-кабель — принтер автоматически переподключится по Wi-Fi.

4. Альтернативные методы подключения

Если стандартная настройка не сработала, попробуйте один из этих способов. Они особенно полезны, если у вас нет доступа к роутеру или принтер упорно не подключается к сети.

Метод 1: Wi-Fi Direct (прямое подключение без роутера)

  • 📱 На принтере удерживайте кнопку Wi-Fi до мигания индикатора.
  • 🖥️ На компьютере откройте список Wi-Fi сетей и найдите сеть с названием CANON-XX-MG3600 (где XX — последние цифры серийного номера).
  • 🔑 Подключитесь к этой сети. Пароль по умолчанию: 12345678 (если не сработает, посмотрите его на наклейке сзади принтера).
  • 🖨️ Запустите печать — принтер будет доступен только пока активно это подключение.

Метод 2: Настройка через приложение Canon PRINT (для смартфонов)

  1. Установите приложение Canon PRINT на смартфон (Android/iOS).
  2. Включите Bluetooth на телефоне и принтере (кнопка Wi-Fi + Чёрный одновременно).
  3. В приложении выберите "Добавить принтер" → "Настройка принтера".
  4. Следуйте инструкциям, чтобы передать настройки Wi-Fi с телефона на принтер.
💡

Если принтер подключён по Wi-Fi Direct, но компьютер его не видит, попробуйте вручную добавить IP-адрес принтера в настройках порта. Для этого откройте "Устройства и принтеры" → добавьте новый принтер → выберите "Добавить принтер по TCP/IP-адресу" и введите IP (узнать его можно через утилиту IJ Network Tool).

Метод 3: Подключение через WPS (если роутер поддерживает)

  1. На роутере нажмите кнопку WPS (обычно она подписана и расположена сзади).
  2. В течение 2 минут на принтере удерживайте кнопку Wi-Fi до мигания индикатора.
  3. Нажмите кнопку Чёрный (или Цветной, в зависимости от прошивки) и удерживайте 5 секунд.
  4. Индикатор Wi-Fi должен загореться зелёным — подключение установлено.
Что делать, если принтер подключился к Wi-Fi, но компьютер его не видит?

1. Перезагрузите оба устройства.

2. В программе IJ Network Tool нажмите "Обновить" — принтер должен появиться в списке.

3. Проверьте, что в настройках принтера включён протокол LPR (в меню "Сетевые настройки" → "Протоколы").

4. Если используется антивирус, временно отключите его и проверьте соединение снова.

5. Настройка принтера на компьютере (Windows/macOS/Linux)

После успешного подключения принтера к Wi-Fi необходимо добавить его в систему. Процесс отличается в зависимости от операционной системы.

Для Windows 10/11:

  1. Откройте "Параметры" → "Устройства" → "Притеры и сканеры".
  2. Нажмите "Добавить принтер или сканер".
  3. В списке найдите Canon MG3640S и выберите его. Если принтера нет — нажмите "Нужный принтер отсутствует в списке" и добавьте его вручную по IP-адресу.
  4. Выберите установленный ранее драйвер и завершите настройку.

Для macOS:

  1. Откройте "Системные настройки" → "Притеры и сканеры".
  2. Нажмите "+" для добавления нового принтера.
  3. Выберите вкладку "IP" и введите адрес принтера (можно узнать через IJ Network Tool).
  4. В поле "Протокол" выберите "Line Printer Daemon - LPD" и укажите очередь печати lp.

Для Linux (Ubuntu/Debian):

  1. Установите пакеты cups и printer-driver-gutenprint через терминал: 
    sudo apt update

    sudo apt install cups printer-driver-gutenprint

  2. Откройте веб-интерфейс CUPS по адресу http://localhost:631.
  3. Перейдите в "Administration" → "Add Printer" и выберите Canon MG3640S из списка сетевых устройств.
  4. Выберите драйвер Canon PIXMA MG3600 Series и завершите настройку.
⚠️ Внимание: На Linux может потребоваться ручная настройка прав доступа к устройству. Если принтер не печатает, добавьте вашего пользователя в группу lpadmin командой sudo usermod -a -G lpadmin ваше_имя_пользователя.

6. Устранение неполадок и частые ошибки

Даже после правильной настройки могут возникать проблемы. Вот самые распространённые ошибки и их решения:

Принтер подключён к Wi-Fi, но не печатает:

  • 🔌 Проверьте, установлен ли принтер как "по умолчанию" в настройках системы.
  • 📑 Убедитесь, что в очереди печати нет застрявших документов (откройте "Очередь печати" и очистите её).
  • 🔄 Перезагрузите службу печати: на Windows выполните в командной строке: 
    net stop spooler

    net start spooler

Притер периодически отключается от Wi-Fi:

  • 📡 Переместите принтер ближе к роутеру (максимальное расстояние — 10 метров без препятствий).
  • 🔄 Измените канал Wi-Fi в настройках роутера (например, с 6 на 11).
  • ⚡ Отключите режим энергосбережения Wi-Fi адаптера в диспетчере устройств Windows.

Ошибка "Невозможно подключиться к принтеру" (код 0x00000002):

  • 🔧 Удалите принтер из системы и добавьте его заново.
  • 🛡️ Временно отключите антивирус и брандмауэр.
  • 📋 Проверьте, что на принтере включён протокол LPR (в меню "Сетевые настройки" → "Протоколы").

Если ни один из способов не помог, попробуйте сбросить настройки принтера до заводских:

  1. Выключите принтер.
  2. Зажмите кнопку Стоп/Сброс (красный треугольник) и удерживайте её, включая принтер.
  3. Удерживайте кнопку 10-15 секунд, пока индикатор не мигнёт 15 раз.
  4. Отпустите кнопку — принтер выполнит сброс.
💡

Большинство проблем с подключением Canon MG3640S связаны с неверными настройками роутера (режим 5 ГГц, скрытый SSID) или отсутствием правильных драйверов. Всегда начинайте диагностику с проверки этих двух пунктов.

7. Оптимизация работы принтера по Wi-Fi

Чтобы избежать проблем в будущем, настройте принтер и сеть оптимальным образом:

Настройки роутера для стабильной работы:

  • 🔒 Задайте статический IP-адрес для принтера в настройках DHCP роутера (например, 192.168.1.100).
  • 📶 Отключите функцию "Автоматический выбор канала" и установите фиксированный канал (например, 1 или 11).
  • ⚡ Включите режим "WMM (Wi-Fi Multimedia)" для приоритизации трафика.

Настройки принтера:

  • 🖨️ В меню принтера отключите "Сон после бездействия" (это предотвратит разрывы соединения).
  • 📄 Установите "Качество печати по умолчанию" на "Стандартное" — это уменьшит размер передаваемых данных.
  • 🔄 Включите "Автоматическое обновление прошивки" (в утилите Canon IJ Network Tool).

Настройки компьютера:

  • 🖥️ В свойствах принтера отключите "Разрешить спаривание устройств" (если не используете Bluetooth).
  • 📂 Очищайте очередь печати раз в месяц, даже если ошибок нет.
  • 🛡️ Добавьте программы Canon в исключения антивируса.

8. Печать с мобильных устройств

Одно из преимуществ Wi-Fi подключения — возможность печатать со смартфонов и планшетов. Для этого:

Для Android:

  1. Установите приложение Canon PRINT из Google Play.
  2. Подключитесь к той же Wi-Fi сети, что и принтер.
  3. В приложении нажмите "Притер" → выберите MG3640S.
  4. Выберите документ или фото для печати и настройте параметры.

Для iOS (iPhone/iPad):

  1. Установите Canon PRINT из App Store.
  2. Включите AirPrint: на принтере перейдите в "Настройки" → "Сетевые настройки" → "AirPrint" → "Вкл.".
  3. В любом приложении (например, "Фото") выберите "Печать" → "Выбрать принтер" → Canon MG3640S.

Для печати с облачных сервисов (Google Drive, Dropbox):

  • 🌐 Откройте документ в веб-версии сервиса.
  • 🖨️ Выберите "Печать" → "Назначение" → "Сохранить в Google Cloud Print" (если настроено).
  • 📱 Или используйте приложение Canon PRINT для прямой печати из облака.
💡

Если принтер не отображается в списке устройств на смартфоне, попробуйте переподключить его к Wi-Fi через приложение Canon PRINT. Иногда это решает проблемы с обнаружением.

FAQ: Ответы на частые вопросы

Можно ли подключить Canon MG3640S к Wi-Fi без компьютера?

Да, это можно сделать прямо с панели управления принтера или через приложение Canon PRINT на смартфоне. Подробная инструкция приведена в разделе 3 (способ 1) и разделе 4 (метод 2).

Принтер подключён к Wi-Fi, но компьютер его не видит. Что делать?

Проверьте следующие моменты:

  1. Установлены ли драйвера (раздел 2).
  2. Включён ли принтер и горит ли зелёный индикатор Wi-Fi.
  3. Найдите IP-адрес принтера через IJ Network Tool и добавьте его вручную (раздел 5).
  4. Отключите брандмауэр и антивирус на время проверки.

Если ничего не помогает, попробуйте подключиться через Wi-Fi Direct (раздел 4, метод 1).

Как сбросить настройки Wi-Fi на принтере?

Чтобы сбросить сетевые настройки:

  1. Нажмите и удерживайте кнопку Wi-Fi (с иконкой антенны) 10 секунд.
  2. Индикатор начнёт мигать оранжевым — отпустите кнопку.
  3. Нажмите кнопку Чёрный (или Цветной) один раз.
  4. Индикатор погаснет и снова замигает — настройки сброшены.

После этого можно настроить Wi-Fi заново.

Какие порты нужно открыть на роутере для работы принтера?

Для стабильной работы Canon MG3640S по Wi-Fi рекомендуется открыть следующие порты:

  • TCP 9100 — для прямой печати.
  • TCP 515 — для протокола LPR.
  • UDP 161 — для SNMP (мониторинг состояния).

Как это сделать:

  1. Зайдите в настройки роутера (обычно по адресу 192.168.1.1).
  2. Найдите раздел "Переадресация портов" или "Port Forwarding".
  3. Добавьте правила для указанных портов, указав IP-адрес принтера.

Можно ли подключить этот принтер к нескольким сетям одновременно?

Нет, Canon MG3640S поддерживает подключение только к одной Wi-Fi сети за раз. Однако вы можете быстро переключаться между сетями:

  1. Нажмите и удерживайте кнопку Wi-Fi до мигания индикатора.
  2. Выберите "Настройка LAN" → "Беспроводная связь".
  3. Выберите другую сеть из списка или введите её параметры вручную.

Также можно использовать Wi-Fi Direct для временного подключения к другому устройству без изменения основных настроек.
