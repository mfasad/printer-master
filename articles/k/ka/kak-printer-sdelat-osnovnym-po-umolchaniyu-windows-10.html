Как установить принтер основным по умолчанию в Windows 10: 5 проверенных способов

Настройка принтера по умолчанию в Windows 10 кажется простой задачей — пока система не начинает игнорировать ваши изменения или автоматически переключаться на другой устройство. Особенно актуальна проблема для пользователей, подключённых к нескольким принтерам одновременно: домашнему HP LaserJet, офисному Canon imageCLASS или виртуальному Microsoft Print to PDF. После обновлений Windows нередко сбрасывает настройки, и документ уходит не туда, куда нужно.

В этой статье — 5 рабочих методов, как сделать принтер основным по умолчанию, включая скрытые настройки реестра и командную строку для опытных пользователей. Мы также разберём, почему Windows 10 может автоматически менять принтер по умолчанию при подключении к новой сети (например, в офисе или кафе) и как это отключить. Если вы столкнулись с ошибкой «Не удалось установить принтер по умолчанию» — в конце статьи есть отдельный раздел с решениями.

1. Стандартный способ: через «Параметры Windows»

Самый очевидный путь — использовать встроенное меню настроек. Он подходит для 90% пользователей и занимает менее минуты. Важно: если принтер не отображается в списке, сначала проверьте его подключение (по USB или Wi-Fi) и установку драйверов.

Откройте меню Пуск → Параметры → Устройства → Принтеры и сканеры. В списке доступных устройств найдите нужный принтер (например, Brother DCP-T720DW или Epson EcoTank ET-2800) и выполните следующие действия:

  • 🖨️ Кликните по названию принтера левой кнопкой мыши, чтобы выделить его.
  • 🔧 Нажмите кнопку «Управление» под списком устройств.
  • ✅ В новом окне выберите пункт «Установить по умолчанию».
  • 🔄 Если кнопка неактивна, обновите страницу (клавиша F5) или перезагрузите компьютер.

После этого рядом с названием принтера появится зелёная галочка и надпись «По умолчанию». Чтобы проверить результат, попробуйте распечатать тестовый документ (например, через Блокнот или Word).

2. Альтернативный метод: через «Панель управления»

Интерфейс классической Панели управления остаётся актуальным даже в Windows 10, особенно для решения проблем с оборудованием. Этот способ полезен, если новый интерфейс «Параметров» не отображает принтер или выдаёт ошибку.

Чтобы открыть панель, нажмите Win + R, введите команду control printers и нажмите Enter. Далее:

  1. В списке принтеров найдите нужное устройство.
  2. Кликните по нему правой кнопкой мыши.
  3. В контекстном меню выберите «Использовать по умолчанию».
  4. Подтвердите действие, если появится запрос UAC (Контроль учётных записей).

Если принтер отсутствует в списке, проверьте:

  • 🔌 Физическое подключение (кабель USB или статус Wi-Fi-соединения).
  • 📥 Установлены ли драйверы (для этого откройте Диспетчер устройств и найдите раздел «Очереди печати»).
  • 🔄 Статус службы печати: нажмите Win + R → введите services.msc → найдите «Диспетчер печати» и убедитесь, что служба запущена.
💡

Если принтер подключён по сети, но не отображается, попробуйте добавить его вручную через Панель управления → Устройства и принтеры → Добавление принтера, указав IP-адрес устройства.

3. Как отключить автоматическое изменение принтера по умолчанию

Windows 10 по умолчанию использует функцию «Разрешить Windows управлять принтером по умолчанию». Это означает, что система может автоматически переключаться на последний использованный принтер — например, если вы подключились к офисной сети и распечатали документ на Xerox WorkCentre, а дома хотите вернуть настройки на Canon PIXMA.

Чтобы отключить эту функцию:

  1. Откройте Параметры → Устройства → Принтеры и сканеры.
  2. Прокрутите страницу вниз и снимите галочку с пункта «Разрешить Windows управлять принтером по умолчанию».
  3. Закройте окно и перезагрузите компьютер.

После этого принтер останется выбранным по умолчанию независимо от сети или последнего использования. Исключение: если устройство будет отключено или возникнет ошибка печати, Windows может временно переключиться на другой принтер.

4. Настройка через реестр Windows (для опытных пользователей)

Если стандартные методы не работают, причиной может быть повреждение системных ключей реестра. Внимание: редактирование реестра требует осторожности — неправильные изменения могут привести к сбоям в работе системы. Перед началом создайте точку восстановления (Панель управления → Восстановление → Настройка восстановления системы).

Чтобы вручную задать принтер по умолчанию через реестр:

  1. Нажмите Win + R, введите regedit и подтвердите запуск.
  2. Перейдите по пути: 
    HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\Windows
  3. В правой части окна найдите параметр «Device» (типа REG_SZ).
  4. Дважды кликните по нему и введите название вашего принтера в формате: 
    \\имя_компьютера\имя_принтера,winspool,Ne00:

    Например: \\DESKTOP-ABC123\HP_LaserJet_M104w,winspool,Ne02:

  5. Также проверьте параметр «UserSelectedDefault» — его значение должно быть 1.
  6. Закройте редактор реестра и перезагрузите ПК.

Если ключ «Device» отсутствует, его можно создать вручную (правый клик → «Создать» → «Строковый параметр»). Для проверки корректности имени принтера откройте Панель управления → Устройства и принтеры и посмотрите свойства устройства.

Как узнать точное имя принтера для реестра?

Откройте командную строку (Win + R → cmd) и введите:

wmic printer get name,portname,shared

В столбце Name будет отображено системное имя устройства, которое нужно использовать в реестре.

5. Использование командной строки или PowerShell

Для автоматизации или устранения неполадок можно использовать команды CMD или PowerShell. Этот метод полезен администраторам, которые настраивают принтеры на нескольких компьютерах, или если графический интерфейс недоступен.

Чтобы установить принтер по умолчанию через командную строку:

  1. Откройте CMD от имени администратора (Win + X → Командная строка (администратор)).
  2. Введите команду для просмотра списка принтеров: 
    wmic printer list brief
  3. Найдите в списке название вашего принтера (например, Brother HL-L2350DW series) и скопируйте его точное имя.
  4. Выполните команду для установки по умолчанию: 
    rundll32 printui.dll,PrintUIEntry /y /n "имя_принтера"

    Пример: rundll32 printui.dll,PrintUIEntry /y /n "Brother HL-L2350DW series"

Для PowerShell используйте следующий скрипт:

$printerName = "имя_вашего_принтера"

$printer = Get-WmiObject -Query "SELECT * FROM Win32_Printer WHERE Name='$printerName'"


$printer.SetDefaultPrinter()

Если команда возвращает ошибку 0x00000000, проверьте:

  • 🔒 Права администратора (запустите консоль от имени админа).
  • 📝 Правильность названия принтера (регистр не важен, но символы должны совпадать).
  • 🔄 Статус службы печати (как проверить — см. раздел 2).

6. Решение проблем: почему принтер не сохраняется по умолчанию

Если после всех манипуляций принтер всё равно сбрасывается или Windows игнорирует настройки, причины могут быть следующими:

Проблема Возможная причина Решение
Настройки сбрасываются после перезагрузки Повреждение реестра или конфликт групповой политики Проверьте ключи реестра (раздел 4) или выполните команду gpupdate /force в CMD
Кнопка «Установить по умолчанию» неактивна Принтер отключён или не готов к печати Проверьте статус устройства в Панели управления и перезапустите службу печати
Windows автоматически меняет принтер Включена опция «Разрешить Windows управлять...» Отключите её (раздел 3) или настройте групповую политику (для Pro-версий)
Ошибка «Не удалось установить принтер по умолчанию» Повреждение драйверов или очереди печати Удалите и переустановите принтер, очистите очередь (services.msc → Диспетчер печати → Остановить/Запустить)

Особое внимание уделите очереди печати: зависшие задания могут блокировать изменение настроек. Чтобы очистить очередь:

  1. Откройте services.msc и остановите службу «Диспетчер печати».
  2. Перейдите в папку C:\Windows\System32\spool\PRINTERS и удалите все файлы.
  3. Запустите службу обратно.
💡

Если проблема сохраняется после всех действий, проверьте компьютер на вирусы (например, с помощью Kaspersky Virus Removal Tool) — некоторые вредоносные программы блокируют изменения системных настроек.

FAQ: Частые вопросы по настройке принтера по умолчанию

Можно ли сделать принтер по умолчанию для конкретной программы (например, Excel или Photoshop)?

Нет, настройка принтера по умолчанию действует на уровне системы для всех приложений. Однако некоторые программы (например, Adobe Acrobat или AutoCAD) позволяют сохранять индивидуальные настройки печати для каждого документа. Проверьте параметры печати в меню Файл → Печать вашей программы.

После обновления Windows 10 принтер по умолчанию сбрасывается. Как это исправить?

Это известная проблема после крупных обновлений (например, Windows 10 22H2). Решения:

  1. Отключите автоматическое управление принтерами (раздел 3).
  2. Обновите драйвер принтера через официальный сайт производителя (не через Центр обновления Windows).
  3. Если проблема сохраняется, используйте метод с редактированием реестра (раздел 4).
Как сделать сетевой принтер основным по умолчанию?

Сетевые принтеры настраиваются так же, как локальные, но сначала убедитесь, что:

  • Притер добавлен в систему через Панель управления → Устройства и принтеры → Добавление принтера.
  • У вас есть права на печать (если принтер защищён паролем или находится в домене).
  • Компьютер и принтер находятся в одной сети (проверьте IP-адрес принтера).

После добавления используйте любой из методов, описанных выше.

Можно ли задать разные принтеры по умолчанию для разных пользователей на одном ПК?

Да, Windows 10 позволяет каждому пользователю устанавливать свой принтер по умолчанию. Настройки привязываются к учётной записи и не конфликтуют. Например, для пользователя «Администратор» можно задать HP OfficeJet, а для гостевой учётки — Canon Selphy.

Почему после установки нового принтера старый автоматически становится основным?

Это связано с функцией «Последний использованный принтер», которая включена по умолчанию. Windows запоминает, на каком устройстве вы печатали в последний раз, и назначает его основным. Чтобы этого избежать:

  1. Отключите опцию Разрешить Windows управлять принтером по умолчанию (раздел 3).
  2. Или вручную верните нужный принтер через Параметры или Панель управления.
💡

Если вы часто переключаетесь между принтерами, создайте ярлыки на рабочем столе с командами для быстрой смены устройства по умолчанию. Например, для принтера Epson L3150 создайте ярлык с целью: rundll32 printui.dll,PrintUIEntry /y /n "Epson L3150 Series".
