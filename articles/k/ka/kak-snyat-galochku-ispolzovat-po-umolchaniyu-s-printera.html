Как снять галочку «Использовать по умолчанию» с принтера: подробный гайд

Принтер, установленный как устройство по умолчанию, автоматически выбирается системой для каждой новой задачи печати. Это удобно, если вы постоянно работаете с одним устройством, но становится проблемой, когда нужно печатать на разных принтерах — например, на домашнем Canon PIXMA для фотографий и на офисном Brother HL-L2350DW для документов. Windows упорно игнорирует ваш выбор и отправляет задания на «дефолтный» принтер, что приводит к пустой трате бумаги, чернил и нервов.

В этой статье разберём, как убрать галочку «Использовать по умолчанию»** с принтера в разных версиях Windows, почему система может самовольно возвращать настройки, и что делать, если нужный принтер пропадает из списка после обновлений. Также рассмотрим нюансы для сетевых и виртуальных принтеров (например, Microsoft Print to PDF), которые часто «застревают» в статусе по умолчанию.

Почему принтер сам становится устройством по умолчанию?

Windows использует алгоритм автоматического управления принтерами (Let Windows manage my default printer), который появился в Windows 10 версии 1803 и сохранился в Windows 11. Система анализирует:

  • 🖨️ Последний использованный принтер — если вы печатали на HP LaserJet Pro M404n вчера, сегодня он может стать «дефолтным».
  • 📡 Сетевую доступность — если принтер Brother DCP-T720DW подключён по Wi-Fi и включён, Windows отдаст ему приоритет перед USB-устройством.
  • 📋 Тип задания — для документов может выбираться монохромный лазерный принтер, а для изображений — цветной струйный.

Проблема в том, что алгоритм не всегда учитывает ваши предпочтения. Например, если вы подключили новый принтер для теста, система может «забыть» про старый, который использовался годами. Или наоборот — после перезагрузки ПК вернётся галочка на принтере, который давно отключён.

Как убрать галочку «Использовать по умолчанию» в Windows 10/11

Самый очевидный способ — снять галочку вручную через Параметры → Устройства → Принтеры и сканеры. Но здесь есть подводные камни:

  1. Откройте Пуск → Параметры → Устройства → Принтеры и сканеры.

  2. Найдите принтер с надписью «По умолчанию»** и кликните по нему.

  3. Нажмите Управление, затем снимите галочку с пункта Использовать по умолчанию.

Если галочка не снимается или возвращается после перезагрузки, причины могут быть следующими:

  • 🔄 Автоматическое управление принтерами включено (см. следующий раздел).
  • 🖥️ Политики группы (в корпоративных сетях) блокируют изменения.
  • 🛠️ Повреждённые драйверы принтера (особенно актуально для Epson L3110 или Xerox Phaser 3020).

Отключение автоматического управления принтерами

Функция Let Windows manage my default printer — главный виновник самовольного возвращения галочки. Чтобы отключить её:

  1. Перейдите в Параметры → Устройства → Принтеры и сканеры.

  2. Вверху страницы отключите ползунок Разрешить Windows управлять моим принтером по умолчанию.

  3. Выберите вручную принтер, который должен быть основным (или оставьте без «дефолтного»).

Важно: после отключения этой функции Windows перестанет «умно» выбирать принтер под конкретное задание. Теперь ответственность за выбор устройства полностью ложится на вас. Если вы печатаете на разных принтерах, придётся выбирать их вручную при каждом задании.

💡

Если ползунка нет — ваша версия Windows устарела. Обновите систему или используйте метод с редактором реестра (см. раздел ниже).

Снятие галочки через реестр (для опытных пользователей)

Если интерфейс Windows не позволяет снять галочку, можно внести изменения в реестр. Будьте осторожны — неверные действия могут привести к сбоям системы.

  1. Нажмите Win + R, введите regedit и подтвердите запуск.

  2. Перейдите по пути: 

    HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\Windows

  3. Найдите параметр LegacyDefaultPrinterMode и установите его значение на 1.

  4. Если параметра нет — создайте его (ПКМ → Создать → Параметр DWORD (32 бита)).

После перезагрузки ПК функция автоматического управления принтерами будет отключена на уровне системы. Этот метод работает даже в корпоративных сетях, где групповые политики блокируют изменения через интерфейс.

Что делать если нет прав на редактирование реестра?

В корпоративных сетях доступ к реестру часто ограничен. Попробуйте:

1. Обратиться в IT-службу с запросом на изменение настроек принтера.

2. Использовать локальный аккаунт администратора (если разрешено).

3. Печатать через веб-интерфейс принтера (если поддерживается).

Проблемы с сетевыми и виртуальными принтерами

Сетевые принтеры (Kyocera Ecosys P3155dn, Ricoh SP 3600SF) и виртуальные (например, Microsoft XPS Document Writer) часто «застревают» в статусе по умолчанию. Причины:

  • 🌐 Нестабильное сетевое подключение — принтер то появляется, то исчезает из списка.
  • 🖼️ Виртуальные принтеры устанавливаются как «дефолтные» при первом использовании (например, после создания PDF).
  • 🔌 Конфликт IP-адресов — если два принтера в сети имеют одинаковый IP.

Решения:

Проблема Действие Пример для модели
Сетевой принтер пропадает Задайте статический IP в настройках роутера HP OfficeJet Pro 9015e
Виртуальный принтер всегда «по умолчанию» Удалите его через Панель управления → Программы → Включение/отключение компонентов Windows Microsoft XPS Document Writer
Конфликт IP Перезагрузите принтер и роутер, проверьте DHCP-настройки Brother MFC-L2710DW
💡

Для сетевых принтеров всегда настраивайте статический IP. Это предотвратит сбои при изменении адресации DHCP и сохранит настройки по умолчанию.

Что делать, если принтер не отображается в списке?

Иногда нужный принтер просто исчезает из Принтеры и сканеры, и снять с него галочку «по умолчанию» невозможно. Причины и решения:

  • 🔌 Физическое отключение — проверьте кабели USB/сетевой кабель, включите принтер в розетку. Для Wi-Fi принтеров (Canon TS3450) переподключитесь к сети.
  • 🖨️ Повреждённые драйверы — удалите принтер через Панель управления → Устройства и принтеры, затем установите заново с официального сайта (support.hp.com, canon.ru).
  • 🛡️ Блокировка антивирусом — временно отключите защиту (например, Kaspersky или ESET) и проверьте обнаружение принтера.

Если принтер по-прежнему не виден:

  1. Запустите Устранение неполадок принтера через Параметры → Обновление и безопасность → Устранение неполадок.

  2. Проверьте службу Диспетчер печати: 

    Win + R → services.msc → найдите "Диспетчер печати" → Запустить/Перезапустить
💡

Для принтеров Epson (например, L3150) перед переустановкой драйверов удалите все задачи из очереди печати через Панель управления → Устройства и принтеры → ПКМ на принтере → Просмотр очереди печати.

Как навсегда закрепить нужный принтер по умолчанию?

Чтобы избежать постоянного возвращения галочки, следуйте этим правилам:

  1. Отключите «умное» управление (см. раздел выше).

  2. Используйте скрипт для автоматического назначения принтера по умолчанию при запуске ПК: 

    @echo off

    rundll32 printui.dll,PrintUIEntry /y /n "Имя вашего принтера"

    Сохраните файл как set_printer.bat и добавьте его в автозагрузку.

  3. Настройте приоритет в групповой политике (для Windows Pro/Enterprise): 

    gpedit.msc → Конфигурация пользователя → Административные шаблоны → Панель управления → Принтеры → Отключить автоматическое управление принтерами

Для корпоративных пользователей: если IT-политика запрещает изменения, запросите у администратора настройку предпочтительного принтера через сервер печати (например, Printix или PaperCut).

FAQ: Частые вопросы

Можно ли снять галочку «по умолчанию» с принтера на Mac?

Да. Откройте Системные настройки → Принтеры и сканеры, выберите принтер и нажмите Установить по умолчанию для другого устройства. На macOS нет автоматического управления, как в Windows.

Почему после обновления Windows принтер снова становится «по умолчанию»?

Обновления могут сбрасывать настройки, особенно если они затрагивают подсистему печати. После установки патчей проверяйте параметр Let Windows manage my default printer и при необходимости отключайте его заново.

Как сделать так, чтобы при печати PDF всегда использовался виртуальный принтер?

В большинстве программ (например, Adobe Acrobat или Foxit Reader) можно выбрать принтер непосредственно в окне печати. Чтобы избежать ошибок, отключите автоматическое управление принтерами в Windows и вручную выбирайте Microsoft Print to PDF при сохранении документов.

Мой принтер HP DeskJet 2700 не сохраняет настройки по умолчанию. Что делать?

Для бюджетных моделей HP характерны проблемы с драйверами. Удалите принтер через Панель управления → Устройства и принтеры, скачайте последнюю версию ПО с официального сайта и установите его в режиме «Установка с параметрами», выбрав опцию Использовать существующий драйвер.

Можно ли назначить разные принтеры по умолчанию для разных программ?

Нет, Windows не поддерживает привязку принтеров к конкретным приложениям на уровне ОС. Однако некоторые программы (например, AutoCAD или Photoshop) позволяют сохранять настройки печати в профилях. Альтернатива — использовать специализированное ПО вроде FinePrint для управления заданиями.

💡

Если вы часто печатаете с разных устройств (ноутбук, планшет, смартфон), настройте облачную печать через Google Cloud Print (для старых устройств) или HP Smart/Canon PRINT. Это позволит выбирать принтер независимо от «дефолтных» настроек Windows.
