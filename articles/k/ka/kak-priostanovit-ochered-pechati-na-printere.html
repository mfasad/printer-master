Как приостановить очередь печати на принтере: инструкции для всех ОС

Застрявшее задание в очереди печати — одна из самых распространённых проблем при работе с принтерами, которая может парализовать работу на часы. Документ не печатается, новые задания не отправляются, а в статусе гордо красуется надпись "Ожидание" или "Ошибка". Причины разнообразны: от банального сбоя драйвера до конфликта с антивирусом, блокирующим печать. Но независимо от первопричины, первое, что нужно сделать — приостановить очередь печати, чтобы разобраться с проблемой без потери данных.

В этой статье вы найдёте актуальные способы приостановки очереди для Windows 10/11, macOS и Linux, а также инструкции для популярных моделей принтеров (HP LaserJet, Canon PIXMA, Epson EcoTank, Brother HL-L2300D). Мы разберём не только стандартные методы через панель управления, но и альтернативные решения — от командной строки до ручного сброса через веб-интерфейс принтера.

⚠️ Внимание: Если принтер подключён по сети, изменения в очереди печати на одном устройстве могут не отразиться на других. Для корпоративных МФУ с общим доступом приостановку лучше выполнять через веб-интерфейс принтера или сервер печати.

1. Приостановка очереди печати в Windows 10 и 11

В операционных системах Windows управление очередью печати осуществляется через Панель управления или Параметры. Способ работает для большинства моделей принтеров, включая HP OfficeJet, Canon imageCLASS и Xerox Phaser. Главное преимущество этого метода — простота и отсутствие необходимости устанавливать дополнительное ПО.

Чтобы приостановить очередь:

  1. Откройте Панель управления (можно через поиск в меню Пуск).
  2. Перейдите в раздел Устройства и принтеры (в Windows 11 путь может выглядеть как Параметры → Bluetooth и устройства → Принтеры и сканеры).
  3. Найдите свой принтер в списке и кликните по нему правой кнопкой мыши.
  4. Выберите пункт Просмотр очереди печати.
  5. В открывшемся окне нажмите Принтер → Приостановить печать.

После приостановки все задания останутся в очереди, но печать прекратится. Это удобно, если нужно временно прервать процесс (например, для замены картриджа) или диагностировать проблему без потери документов.

⚠️ Внимание: Если кнопка Приостановить печать неактивна, проверьте права пользователя. В корпоративных сетях администраторы часто ограничивают доступ к управлению очередью через групповую политику.

Альтернативный способ — через Диспетчер задач:

  • 🔹 Нажмите Ctrl + Shift + Esc, чтобы открыть Диспетчер.
  • 🔹 Найдите процесс spoolsv.exe (служба очереди печати).
  • 🔹 Кликните правой кнопкой и выберите Завершить задачу.
  • 🔹 Перезапустите службу вручную через services.msc (введите в поиске).

2. Командная строка: быстрая приостановка без интерфейса

Для опытных пользователей или системных администраторов удобнее управлять очередью через командную строку или PowerShell. Этот метод особенно полезен, если принтер подключён к серверу или нужно автоматизировать процесс (например, через скрипт).

Основные команды для работы с очередью: 

net stop spooler & net start spooler  # Перезапуск службы печати

printui /s /t2                       # Открытие окна очереди для указанного принтера

Чтобы приостановить печать для конкретного принтера:

  1. Узнайте точное имя принтера через команду: 
    wmic printer get name
  2. Приостановите очередь: 
    printui /Xs /n "Имя_принтера" Attribute 2048

    (где 2048 — код для приостановки).

💡

Если команда printui не работает, проверьте наличие файла printui.dll в папке C:\Windows\System32. В редких случаях он может отсутствовать после обновлений системы.

Для возобновления печати используйте: 

printui /Xs /n "Имя_принтера" Attribute 0

Этот метод подходит для Windows Server и корпоративных сред, где графический интерфейс может быть отключён. Также он полезен, если очередь "зависла" и не реагирует на стандартные действия.

3. Приостановка очереди на macOS (Ventura, Sonoma)

В macOS управление очередью печати интегрировано в Центр печати (Print Center). Интерфейс минималистичный, но функциональный. Главное отличие от Windows — здесь нет отдельной службы печати, поэтому сбои случаются реже, но и инструментов для диагностики меньше.

Пошаговая инструкция:

  • 🔹 Откройте Центр печати через Системные настройки → Принтеры и сканеры.
  • 🔹 Выберите принтер и нажмите Открыть очередь печати.
  • 🔹 В верхнем меню кликните Принтер → Приостановить печать.
  • 🔹 Для полной очистки очереди используйте Принтер → Очистить очередь.

Если очередь не реагирует:

  1. Закройте Центр печати.
  2. Откройте Терминал и введите: 
    cancel -a -

    (это удалит все задания для текущего пользователя).

  3. Перезапустите службу печати: 
    sudo launchctl stop com.apple.printtool.plist

    sudo launchctl start com.apple.printtool.plist

Что делать, если принтер не отображается в macOS?

Если принтер исчез из списка, попробуйте удалить его и добавить заново через Системные настройки → Принтеры и сканеры → "+". Если проблема сохраняется, проверьте совместимость драйвера с вашей версией macOS (особенно актуально для старых моделей Epson Stylus или Canon LBP).

Для сетевых принтеров (например, Brother HL-L8360CDW) может потребоваться приостановка через веб-интерфейс устройства. Адрес обычно имеет вид http://[IP-адрес_принтера].

4. Управление очередью в Linux (Ubuntu, Debian, Fedora)

В Linux-системах очередь печати управляется через CUPS (Common Unix Printing System). Это гибкая система, но требует знания команд или работы с веб-интерфейсом. Способ подходит для принтеров Kyocera, Ricoh и других, поддерживаемых открытыми драйверами.

Базовые команды для приостановки: 

lpstat -p -d               # Просмотр всех принтеров и статусов

cupsdisable [имя_принтера] # Приостановка принтера


cupsenable [имя_принтера]  # Возобновление печати

Чтобы приостановить конкретное задание: 

lp -i [номер_задания] -H hold  # Приостановить

lp -i [номер_задания] -H release # Возобновить

Номер задания можно узнать через: 

lpstat -o

Для полной очистки очереди: 

cancel -a -

Если принтер подключён по сети, может потребоваться доступ к его веб-интерфейсу. Например, для HP Color LaserJet M454 адрес будет http://[IP-принтера]/hp/device/this.LCDispatcher?nav=hp-PrintJobs.

5. Приостановка через веб-интерфейс принтера

Современные сетевые принтеры (например, Xerox VersaLink, Canon imageRUNNER) оснащены встроенным веб-сервером, который позволяет управлять очередью напрямую. Это особенно удобно в офисах, где к одному устройству подключены несколько компьютеров.

Как найти веб-интерфейс:

  • 🔹 Напечатайте сетевой отчёт (обычно кнопка i или Network на панели принтера).
  • 🔹 В отчёте будет указан IP-адрес (например, 192.168.1.100).
  • 🔹 Введите этот адрес в браузере (рекомендуется Chrome или Firefox).

В интерфейсе найдите раздел "Очередь заданий" или "Print Jobs". Здесь можно:

  • 🔹 Приостановить отдельные задания (кнопка Pause или Hold).
  • 🔹 Удалить конкретные документы.
  • 🔹 Изменить приоритет печати.

💡

Веб-интерфейс принтера — единственный способ управлять очередью, если принтер подключён к сетевому серверу печати (например, в крупных компаниях). Локальные действия на ПК в этом случае не помогут.

Для принтеров Brother путь может выглядеть так: Сеть → Очередь заданий → Управление

Для HP Enterprise: Устройство → Очередь печати → Действия

⚠️ Внимание: Некоторые производители (например, Epson) блокируют доступ к веб-интерфейсу в базовых моделях. В этом случае управлять очередью придётся через ПК.

6. Частые ошибки и их решения

Приостановка очереди — только первый шаг. Часто проблема кроется глубже. Вот типичные сценарии и способы их решения:

Симптом Возможная причина Решение
Очередь не приостанавливается Сбой службы spoolsv.exe (Windows) Перезапустите службу через services.msc или командную строку
Документ висит в статусе "Удаление" Конфликт с драйвером принтера Удалите принтер и добавьте заново с официального драйвера
После приостановки печать не возобновляется Зависание задания в статусе Processing Очистите очередь через net stop spooler & del %systemroot%\System32\spool\PRINTERS\* /Q & net start spooler
Приостановка работает, но принтер всё равно печатает Принтер имеет собственную память заданий Отключите принтер от сети на 30 секунд для сброса буфера

Если проблема повторяется регулярно, проверьте:

  • 🔹 Драйвер принтера — скачайте актуальную версию с сайта производителя.
  • 🔹 Антивирус — некоторые программы (например, Kaspersky) блокируют службу печати.
  • 🔹 Сетевые настройки — для Wi-Fi принтеров проверьте стабильность соединения.

💡

Для принтеров HP с ошибкой 49.4C02 (зависание очереди) поможет сброс настроек через меню Service → Restore Defaults (доступно в сервисном режиме).

7. Особенности для популярных моделей принтеров

Некоторые принтеры имеют уникальные нюансы управления очередью. Рассмотрим наиболее распространённые модели:

HP LaserJet (серии M400, M500, Enterprise):

  • 🔹 Поддерживают веб-интерфейс с расширенными настройками очереди.
  • 🔹 Для сброса зависших заданий используйте Cold Reset (удерживайте кнопку включения 10 секунд).

Canon PIXMA (MG, TS, G серии):

  • 🔹 Часто возникают проблемы с драйверами при обновлении Windows.
  • 🔹 Для очистки очереди используйте утилиту Canon PRINT Utility (скачать с официального сайта).

Epson EcoTank (L3250, L805, ET-2800):

  • 🔹 В веб-интерфейсе отсутствует функция приостановки — только удаление заданий.
  • 🔹 Для сброса буфера печати отключите принтер от розетки на 1 минуту.

Brother HL-L2300D, DCP-L2550DW:

  • 🔹 Поддерживают BRAdmin Light — утилиту для управления сетевыми принтерами.
  • 🔹 При зависании очереди поможет перепрошивка через Firmware Update Tool.

⚠️ Внимание: Для принтеров Xerox WorkCentre приостановка через веб-интерфейс может потребовать аутентификации (логин/пароль по умолчанию часто admin/1111 или admin/2222).

FAQ: Частые вопросы по управлению очередью печати

Можно ли приостановить печать только для одного документа, не затрагивая остальные?

Да, в Windows и macOS можно приостановить отдельное задание. В очереди печати кликните правой кнопкой по документу и выберите Приостановить. В Linux используйте команду lp -i [номер_задания] -H hold.

Почему после приостановки принтер всё равно печатает?

Это происходит, если принтер имеет собственную память (буфер). Решение:

  1. Отключите принтер от сети на 30–60 секунд.
  2. Используйте веб-интерфейс принтера для очистки его внутренней очереди.

Как очистить очередь, если она не реагирует на команды?

Для Windows: 

net stop spooler

del %systemroot%\System32\spool\PRINTERS\* /Q


net start spooler

Для macOS:

sudo rm /var/spool/cups/*

Для Linux:

sudo systemctl stop cups

sudo rm -rf /var/spool/cups/*


sudo systemctl start cups

Можно ли восстановить документ после очистки очереди?

Нет, после очистки данные удаляются безвозвратно. Если документ важен, перед очисткой очереди сохраните его исходный файл (например, PDF или Word) на компьютере.

Почему очередь печати постоянно зависвает?

Частые причины:

  • 🔹 Устаревший или повреждённый драйвер.
  • 🔹 Конфликт с антивирусом (добавьте исключение для spoolsv.exe).
  • 🔹 Нехватка памяти на принтере (актуально для старых моделей Samsung или Lexmark).
  • 🔹 Повреждённые файлы в папке C:\Windows\System32\spool\PRINTERS.
