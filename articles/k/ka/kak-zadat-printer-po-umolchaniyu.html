Как сделать принтер основным по умолчанию: инструкции для Windows 10/11 и macOS

Принтер по умолчанию — это устройство, на которое автоматически отправляются документы при нажатии кнопки «Печать». Без правильной настройки системы могут возникать задержки, ошибки или печать на не том устройстве — особенно в офисах с несколькими МФУ. Эта статья поможет разобраться, как задать принтер по умолчанию на разных платформах, исправить типичные ошибки и оптимизировать процесс печати.

Мы рассмотрим не только базовые инструкции для Windows 10/11 и macOS, но и нюансы работы с сетевыми принтерами, виртуальными устройствами (например, PDF-принтерами) и драйверами. Отдельное внимание уделим проблемам, когда система «забывает» настройки после перезагрузки или обновлений.

Если вы работаете с сканерами штрих-кодов, МФУ или специализированными печатающими устройствами (например, для этикеток), приведённые ниже методы помогут избежать конфликтов драйверов и сбоев в работе оборудования.

Почему важно настроить принтер по умолчанию

Без явного указания основного принтера система может выбирать устройство случайно — особенно если к компьютеру подключено несколько МФУ или виртуальных принтеров (например, Microsoft Print to PDF или OneNote). Это приводит к:

  • 📄 Потере времени — документы уходят не на тот принтер, и их приходится перепечатывать.
  • 💰 Перерасходу ресурсов — в офисах с платной печатью (например, через Follow-Me Printing) это ведёт к лишним тратам.
  • 🔄 Сбоям в работе ПО — некоторые программы (например, 1С или AutoCAD) требуют явного указания принтера для корректной печати.
  • 🖨️ Конфликтам драйверов — если система постоянно переключается между устройствами, это может вызвать ошибки spooler’а (службы печати).

В Windows 10/11 и macOS алгоритм выбора принтера по умолчанию различается. Например, в Windows система может автоматически назначать последнее использованное устройство, а в macOS — приоритезировать принтер, подключённый по AirPrint. Разберёмся, как управлять этим процессом.

Как задать принтер по умолчанию в Windows 10 и 11

В последних версиях Windows Microsoft изменила логику назначения принтера по умолчанию. Теперь система по умолчанию использует функцию «Управление принтером по умолчанию» (Let Windows manage my default printer), которая автоматически выбирает последнее использованное устройство. Это удобно для пользователей с одним принтером, но создаёт хаос в офисах. Вот как отключить эту функцию и задать принтер вручную.

Шаг 1. Отключение автоматического управления принтером

Перейдите в Параметры → Устройства → Принтеры и сканеры. В верхней части окна найдите переключатель «Разрешить Windows управлять принтером по умолчанию» и отключите его.

После этого система перестанет автоматически менять принтер, и вы сможете назначить основное устройство вручную.

Шаг 2. Выбор принтера по умолчанию

В том же разделе Принтеры и сканеры найдите нужное устройство в списке, кликните на него и выберите «Управление». В открывшемся меню нажмите «Назначить по умолчанию».

Если кнопка неактивна, это может означать:

  • 🔌 Принтер отключён от сети или компьютера.
  • 🚫 Устройство не установлено корректно (отсутствуют драйверы).
  • 🔄 Служба печати (Print Spooler) остановлена.

Убедитесь, что принтер включён и подключён к сети|Проверьте статус службы Print Spooler (должна быть запущена)|Обновите драйверы принтера|Перезагрузите компьютер после изменений-->

Если принтер не отображается в списке, попробуйте добавить его вручную через Добавить принтер или сканер. Для сетевых устройств может потребоваться ввод IP-адреса или выбора из Active Directory (в корпоративных сетях).

Шаг 3. Проверка через панель управления (альтернативный метод)

Некоторые настройки проще выполнить через классическую Панель управления. Для этого:

  1. Откройте Панель управления → Устройства и принтеры.
  2. Найдите нужный принтер, кликните по нему правой кнопкой мыши.
  3. Выберите «Назначить принтером по умолчанию» (значок принтера должен стать зелёным с галочкой).

Если в этом меню принтер отсутствует, попробуйте обновить список устройств через Обновить или переустановить драйвер.

💡

Если принтер по умолчанию сбрасывается после перезагрузки, проверьте групповую политику (для Windows Pro/Enterprise). В редакторе gpedit.msc перейдите в Конфигурация пользователя → Административные шаблоны → Панель управления → Принтеры и отключите политику автоматического управления.

Настройка принтера по умолчанию в macOS (Ventura, Sonoma и старше)

В macOS процесс назначения принтера по умолчанию проще, но есть свои нюансы — особенно при работе с AirPrint или сетевыми устройствами. Рассмотрим пошаговую инструкцию для последних версий системы.

Шаг 1. Открытие настроек принтера

Перейдите в Системные настройки → Принтеры и сканеры (в старых версиях — Системные предпочтения). В левой части окна отобразится список установленных принтеров.

Если нужного устройства нет, нажмите «+» внизу списка и выберите:

  • 🖨️ Локальный принтер — для устройств, подключённых по USB.
  • 🌐 Сетевой принтер — для МФУ в офисной сети (может потребоваться ввод IP или выбора из Bonjour).
  • ☁️ AirPrint — для беспроводных принтеров Apple или совместимых моделей (например, HP LaserJet, Canon PIXMA).

Шаг 2. Назначение принтера по умолчанию

Выделите нужный принтер в списке и кликните «Назначить принтером по умолчанию» (или «Set as Default Printer» в английской версии). В macOS Sonoma эта опция может находиться в контекстном меню (клик правой кнопкой по принтеру).

Если опция неактивна, проверьте:

  • 🔌 Состояние подключения (для USB — кабель, для сети — Wi-Fi/Ethernet).
  • 📦 Наличие драйверов (для некоторых моделей, например, Brother или Epson, требуется установка ПО с официального сайта).
  • 🔄 Статус очереди печати (зависшие задачи могут блокировать настройки).

Шаг 3. Дополнительные настройки (опционально)

Для тонкой настройки перейдите в «Параметры принтера» (кнопка «Options & Supplies») и проверьте:

  • 📇 Имя принтера — должно быть уникальным в сети.
  • 🖼️ Разрешение печати (например, 600 dpi или 1200 dpi).
  • 📄 Формат бумаги (особенно важно для этикеточных принтеров, например, Zebra или Dymo).

В macOS также можно настроить предпочтительный принтер для конкретных приложений. Например, в Adobe Acrobat или Microsoft Word можно явным образом указать устройство для печати, игнорируя системные настройки.

Как сбросить настройки принтера в macOS?

Если принтер работает некорректно, попробуйте сбросить систему печати:

1. Откройте Терминал (Applications → Utilities → Terminal).

2. Введите команду: 

sudo cupsctl WebInterface=no

3. Перезагрузите компьютер.

4. Зайдите вновь в Системные настройки → Принтеры и сканеры и добавьте принтер заново.

Решение проблем: принтер по умолчанию сбрасывается или не сохраняется

Одна из самых распространённых проблем — система «забывает» принтер по умолчанию после перезагрузки, обновления или выхода из сна. Это может быть связано с:

Причина Признаки Решение
Автоматическое управление в Windows Принтер меняется после каждого использования другого устройства Отключить опцию Let Windows manage my default printer
Конфликт драйверов Принтер исчезает из списка или появляется с восклицательным знаком Удалить все драйверы принтера и установить заново с официального сайта
Служба Print Spooler остановлена Ошибка «Не удалось сохранить параметры принтера» Запустить службу через services.msc или командой 
net start spooler
Групповая политика (в корпоративных сетях) Настройки сбрасываются при входе в домен Обратиться к администратору для изменения политики Point and Print Restrictions
Обновление macOS Принтер исчезает после апгрейда системы Добавить принтер заново через Системные настройки

Если проблема сохраняется, попробуйте сбросить настройки печати:

  • В Windows: удалите все принтеры из Устройства и принтеры, очистите очередь печати (C:\Windows\System32\spool\PRINTERS) и перезагрузите ПК.
  • В macOS: выполните сброс CUPS (см. спойлер выше).
💡

Если принтер по умолчанию сбрасывается только для конкретного пользователя, проблема может быть в повреждённом профиле. Создайте нового пользователя в системе и проверьте, сохраняются ли настройки.

Особенности работы с сетевыми и виртуальными принтерами

Сетевые принтеры (например, HP OfficeJet Pro или Xerox WorkCentre) и виртуальные устройства (например, PDF-принтеры или Google Cloud Print) требуют отдельного внимания. Их настройка как принтера по умолчанию имеет нюансы.

Сетевые принтеры

Для назначения сетевого принтера по умолчанию:

  1. Убедитесь, что принтер доступен в сети (проверьте ping по IP-адресу).
  2. Добавьте его через Добавить принтер → Добавить по TCP/IP-адресу (в Windows) или IP-принтер (в macOS).
  3. При запросе драйвера выберите точную модель из списка или установите с диска производителя.

В офисных сетях принтер может быть привязан к Active Directory. В этом случае его настройка управляется групповой политикой, и изменить принтер по умолчанию может только администратор.

Виртуальные принтеры (PDF, XPS, OneNote)

Виртуальные принтеры (например, Microsoft Print to PDF или CutePDF) также можно назначить по умолчанию. Это полезно, если вы часто сохраняете документы в электронном виде. Однако учтите:

  • 📂 Файлы будут сохраняться в папку по умолчанию (обычно Документы или Downloads).
  • 🔄 Некоторые программы (например, AutoCAD) могут игнорировать виртуальные принтеры при печати.
  • 🛠️ Для работы CutePDF или doPDF требуется установка дополнительного ПО.

Чтобы назначить виртуальный принтер по умолчанию, следуйте той же инструкции, что и для физического устройства. В macOS для этих целей часто используется Preview или Command + P → Save as PDF.

💡

Если виртуальный принтер не отображается в списке, проверьте, установлено ли ПО для него. Например, для FinePrint или PDF24 требуется отдельная установка.

Как проверить, какой принтер установлен по умолчанию

Иногда пользователи не уверены, какой принтер в данный момент является основным. Проверить это можно несколькими способами.

В Windows

  • 🖥️ Откройте Параметры → Устройства → Принтеры и сканеры. Принтер по умолчанию будет отмечен надписью «По умолчанию».
  • 📋 В Панели управления → Устройства и принтеры иконка принтера по умолчанию имеет зелёную галочку.
  • 💻 Через командную строку: выполните команду 
    wmic printer get name,default
    — принтер с TRUE в колонке Default является текущим.

В macOS

  • 🍎 Откройте Системные настройки → Принтеры и сканеры. Принтер по умолчанию будет выделен жирным шрифтом.
  • 📄 В диалоговом окне печати (Command + P) текущий принтер отображается в выпадающем меню.

Если принтер по умолчанию не отображается или отмечен восклицательным знаком, это указывает на проблемы с драйвером или подключением. Попробуйте перезагрузить устройство или компьютер.

Дополнительные советы по оптимизации печати

Настройка принтера по умолчанию — только часть процесса оптимизации печати. Вот несколько рекомендаций, которые помогут избежать проблем:

  • 🔄 Регулярно обновляйте драйверы — особенно для МФУ (например, Kyocera или Ricoh), где новые версии ПО исправляют ошибки печати.
  • 📡 Используйте статический IP для сетевых принтеров — это предотвратит потерю связи после перезагрузки роутера.
  • 📁 Очищайте очередь печати — зависшие задачи могут блокировать новые. В Windows это делается через services.msc (остановка и запуск Print Spooler).
  • 🔒 Настройте права доступа — в офисах ограничьте возможность изменения принтера по умолчанию для обычных пользователей.

Для специализированных устройств (например, сканеров штрих-кодов с функцией печати или термопринтеров) может потребоваться ручная настройка портов (например, LPT1 или COM) или установка фирменного ПО (например, ZebraDesigner для принтеров этикеток).

💡

Если вы работаете с термотрансферными принтерами (например, Zebra ZD420), никогда не назначайте их принтерами по умолчанию для обычных документов. Эти устройства оптимизированы для печати этикеток и могут искажать текстовые файлы.

FAQ: Частые вопросы о настройке принтера по умолчанию

Можно ли назначить принтер по умолчанию для конкретной программы?

Да, некоторые приложения (например, Microsoft Office или Adobe Acrobat) позволяют переопределять системные настройки печати. Для этого при печати (Ctrl + P) выберите нужный принтер вручную и сохраните его как предпочтительный для данного документа.

Почему принтер по умолчанию сбрасывается после обновления Windows?

Это связано с тем, что крупные обновления (например, Windows 11 23H2) сбрасывают некоторые пользовательские настройки. Чтобы избежать проблемы, отключите автоматическое управление принтерами (см. раздел выше) и назначьте устройство заново после апгрейда.

Как сделать принтер по умолчанию в Linux (Ubuntu, Mint)?

В большинстве дистрибутивов Linux настройка выполняется через CUPS (Common Unix Printing System). Откройте в браузере http://localhost:631, перейдите в Printers, выберите устройство и нажмите «Set As Default». Альтернативно можно использовать команду в терминале: 

lpoptions -d PRINTER_NAME

Что делать, если принтер по умолчанию не сохраняется в домене?

В корпоративных сетях настройки принтера могут управляться групповой политикой. Обратитесь к администратору, чтобы он провёл настройку через Group Policy Management (gpmc.msc) или разрешил пользователям изменять принтер по умолчанию через политику Point and Print Restrictions.

Можно ли назначить принтер по умолчанию через командную строку?

Да, в Windows это делается командой: 

rundll32 printui.dll,PrintUIEntry /y /n "Имя принтера"

В macOS используйте: 

lpoptions -d PRINTER_NAME

Замените PRINTER_NAME на точное имя устройства (с пробелами в кавычках).

Важно: Если вы используете облачные принтеры (например, Google Cloud Print или HP ePrint), их настройка как устройства по умолчанию может потребовать авторизации в аккаунте и стабильного интернет-соединения. В противном случае печать будет прерываться.

⚠️ Внимание: Настройки групповой политики и драйверов принтеров могут отличаться в зависимости от версии ОС и модели устройства. Для корпоративных МФУ (например, Xerox AltaLink или Canon imageRUNNER) всегда сверяйтесь с документацией производителя или настройками в Active Directory.
