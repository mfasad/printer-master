Как убрать значок принтера по умолчанию: инструкция для Windows, macOS и Linux

Значок принтера с галочкой или звездочкой — стандартный индикатор устройства по умолчанию в операционных системах. Он появляется автоматически после установки нового оборудования или смены приоритетного устройства печати. Однако не всем пользователям удобно видеть этот маркер: он может мешать восприятию рабочего стола, создавать путаницу при работе с несколькими принтерами или просто раздражать своим наличием.

В этой статье мы разберём 5 рабочих способов удаления значка принтера по умолчанию — от простых настроек ОС до редактирования реестра. Инструкции актуальны для Windows 10/11, macOS Ventura/Monterey и дистрибутивов Linux (Ubuntu, Debian, Fedora). Также вы узнаете, почему значок может появляться повторно и как это предотвратить.

Важно: если вы используете корпоративный принтер с централизованным управлением (например, через Active Directory или CUPS), некоторые методы могут быть заблокированы администратором. В этом случае обратитесь в IT-службу.

Почему появляется значок принтера по умолчанию

Операционные системы автоматически назначают принтер по умолчанию для удобства пользователя. Этот статус отображается визуально:

  • 🖨️ Windows: зелёная галочка на иконке принтера в панели управления или параметрах
  • 🍎 macOS: жирный шрифт в списке принтеров или значок "Default" в Системные настройки → Принтеры и сканеры
  • 🐧 Linux: звездочка (*) рядом с названием в CUPS или Настройки → Принтеры

Основные причины появления значка:

  1. Автоматическое назначение — ОС выбирает последний установленный или наиболее часто используемый принтер.
  2. Политика группы — в корпоративных сетях администраторы принудительно задают принтер по умолчанию.
  3. Обновление драйверов — после переустановки ПО принтер может снова стать основным.
  4. Сброс настроек — после обновления Windows или macOS настройки печати возвращаются к заводским.

В большинстве случаев значок — это просто визуальный маркер, не влияющий на работу устройства. Но если он мешает, его можно убрать.

Способ 1: Снятие статуса "По умолчанию" в настройках Windows

Самый простой метод для Windows 10/11 — изменить принтер по умолчанию на другой или отключить автоматическое назначение. Следуйте инструкции:

  1. Откройте Параметры → Устройства → Принтеры и сканеры (или Панель управления → Оборудование и звук → Устройства и принтеры).
  2. Найдите принтер со значком галочки и кликните на него.
  3. Выберите Управление (или правой кнопкой → Назначить принтером по умолчанию).
  4. Снимите галочку с опции Разрешить Windows управлять принтером по умолчанию.
  5. Вручную выберите другой принтер (например, Microsoft Print to PDF) или оставьте без выбора.

Если после этих действий значок остался, перезагрузите компьютер. В 90% случаев это решает проблему.

⚠️ Внимание: В некоторых версиях Windows (например, Windows 10 LTSC) опция автоматического управления принтерами может отсутствовать. В этом случае используйте Способ 3 (редактирование реестра).

Способ 2: Удаление значка через реестр Windows

Если стандартные настройки не помогают, можно вручную отредактировать реестр. Этот метод подходит для опытных пользователей.

Перед началом обязательно создайте точку восстановления системы:

  1. Нажмите Win + R, введите regedit и подтвердите запуск.
  2. Перейдите по пути: 
    HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\Windows
  3. Найдите параметр LegacyDefaultPrinterMode (если его нет — создайте DWORD (32-bit) с таким именем).
  4. Установите значение 1 и перезагрузите ПК.

После перезагрузки:

  • 🔄 Значок галочки исчезнет, но принтер останется рабочим.
  • 🔧 При необходимости верните настройки, удалив параметр или установив значение 0.
Параметр реестра Значение Эффект
LegacyDefaultPrinterMode 0 Автоматическое управление принтерами (значок остаётся)
LegacyDefaultPrinterMode 1 Ручной выбор принтера (значок исчезает)
DeviceNotSelectedTimeout 15 Время ожидания перед сбросом принтера (в секундах)
⚠️ Внимание: Неправильное редактирование реестра может привести к сбоям в работе системы. Если не уверены в своих действиях, используйте Способ 4 (групповые политики).
Что делать если реестр заблокирован администратором?

Если при попытке изменить реестр вы видите сообщение "Изменение запрещено администратором", значит на вашем ПК действуют групповой политики домена. В этом случае:

1. Обратитесь в IT-отдел с запросом на разблокировку.

2. Попробуйте временно отключиться от корпоративной сети (если разрешено).

3. Используйте локальную учётную запись вместо доменной (если доступно).

Способ 3: Использование групповой политики (для Windows Pro/Enterprise)

В корпоративных версиях Windows (Pro, Enterprise, Education) можно отключить автоматическое назначение принтера через Редактор локальной групповой политики.

Инструкция:

  1. Нажмите Win + R, введите gpedit.msc и подтвердите.
  2. Перейдите по пути: Конфигурация пользователя → Административные шаблоны → Панель управления → Принтеры
  3. Найдите политику Всегда устанавливать принтер по умолчанию и откройте её.
  4. Установите переключатель в положение Отключено и примените изменения.

После применения политики:

  • 📌 Значок галочки исчезнет после перезагрузки.
  • 🔒 Настройки будут действовать для всех пользователей ПК.
  • 🔄 При обновлении Windows политику может потребоваться применить заново.
Get-AppxPackage Microsoft.Windows.GroupPolicy | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

После перезагрузки gpedit.msc станет доступен.-->

Способ 4: Удаление значка в macOS

В macOS процесс немного отличается. Здесь значок "По умолчанию" отображается в виде жирного шрифта или метки Default в списке принтеров.

Чтобы убрать его:

  1. Откройте Системные настройки → Принтеры и сканеры.
  2. Выделите принтер с меткой Default.
  3. Кликните по кнопке − (минус) внизу списка, чтобы удалить принтер.
  4. Добавьте принтер заново, нажав + → выберите его из списка или добавьте вручную по IP.
  5. При добавлении не отмечайте галочку "Использовать по умолчанию".

Если принтер нужен для печати, но вы не хотите видеть метку:

  • 🖱️ Кликните правой кнопкой на принтере → Сделать принтером по умолчанию (снимите галочку).
  • 🔄 Перезагрузите CUPS (систему печати macOS) через Терминал: 
    sudo launchctl stop org.cups.cupsd

    sudo launchctl start org.cups.cupsd

⚠️ Внимание: В macOS Ventura 13.3+ Apple изменила логику работы с принтерами по умолчанию. Если после сброса метка появляется снова, проверьте настройки в Терминале командой: lpoptions -d — она покажет текущий принтер по умолчанию. Чтобы сбросить, выполните: lpoptions -d NULL

Способ 5: Решение для Linux (CUPS)

В дистрибутивах на базе Linux (Ubuntu, Debian, Fedora) за печать отвечает система CUPS (Common Unix Printing System). Убрать значок принтера по умолчанию можно через веб-интерфейс или терминал.

Метод 1: Через веб-интерфейс CUPS

  1. Откройте браузер и перейдите по адресу: http://localhost:631
  2. Выберите вкладку Printers (Принтеры).
  3. Найдите принтер со звездочкой (*) и кликните на него.
  4. В меню Administration выберите Set As Server Default (Установить как сервер по умолчанию) — снимите эту настройку.
  5. Сохраните изменения и перезагрузите CUPS: 
    sudo systemctl restart cups

Метод 2: Через терминал

Выполните команду для сброса принтера по умолчанию:

lpoptions -d NULL

Чтобы проверить текущий статус, используйте:

lpstat -d

Если значок остался, удалите и добавьте принтер заново:

lpadmin -x PRINTER_NAME  # удаление

lpadmin -p PRINTER_NAME -E -v URI -m DRIVER  # добавление

Где PRINTER_NAME — имя вашего принтера, URI — путь (например, usb://HP/DeskJet%202130), DRIVER — модель драйвера.

💡

В Linux значок принтера по умолчанию управляется через CUPS. Если после сброса он появляется снова, проверьте автозагрузку служб командой systemctl list-units --type=service | grep cups.

Что делать, если значок появляется снова

Если после всех манипуляций значок принтера по умолчанию возвращается, причины могут быть следующими:

Причина Решение
Автообновление драйверов Отключите автоматическое обновление в Параметры → Обновление и безопасность → Дополнительные параметры.
Корпоративные политики Обратитесь в IT-отдел или используйте локальную учётную запись.
Сбой службы печати Перезапустите службу: 
net stop spooler & net start spooler
(Windows) или sudo systemctl restart cups (Linux).
Вирусное ПО Проверьте систему антивирусом (например, Kaspersky Virus Removal Tool).

Если проблема сохраняется:

  • 🔍 Проверьте планировщик задач на наличие автоматических скриптов, меняющих принтер по умолчанию (taskschd.msc в Windows).
  • 📋 Удалите принтер полностью и установите заново с официального сайта производителя.
  • 🖥️ Создайте нового пользователя в системе — возможно, проблема в профиле.
Как проверить, какое ПО меняет принтер по умолчанию?

Используйте утилиту Process Monitor (от Microsoft) для мониторинга изменений в реестре:

1. Скачайте ProcMon с официального сайта.

2. Запустите с фильтром по пути HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\Windows.

3. Отследите, какой процесс изменяет параметр LegacyDefaultPrinterMode.

FAQ: Частые вопросы

Можно ли убрать значок принтера по умолчанию без прав администратора?

Нет, для изменения настроек принтера требуются права администратора. Если у вас ограниченная учётная запись, обратитесь к администратору системы или попробуйте:

  • Использовать Microsoft Print to PDF как принтер по умолчанию (не требует прав).
  • Создать нового пользователя с правами администратора (если разрешено).
Почему после обновления Windows значок появляется снова?

Обновления Windows иногда сбрасывают пользовательские настройки печати. Чтобы это предотвратить:

  1. Перед обновлением экспортируйте раздел реестра HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\Windows.
  2. После обновления импортируйте его обратно.
  3. Используйте скрипт для автоматического применения настроек (пример для PowerShell): 
    Set-ItemProperty -Path "HKCU:\Software\Microsoft\Windows NT\CurrentVersion\Windows" -Name "LegacyDefaultPrinterMode" -Value 1
Как убрать значок на сетевом принтере в офисе?

В корпоративной сети настройки принтера по умолчанию обычно управляются через Active Directory или Print Server. Самостоятельно убрать значок не получится, но можно:

  • Попросить администратора добавить вас в исключение политики.
  • Использовать локальный принтер (например, PDF-принтер) как основной.
  • Подключить принтер напрямую по IP, минуя сервер печати.
⚠️ Внимание: Изменение сетевых настроек без разрешения может нарушить работу офисного оборудования.
Есть ли программы для автоматического управления значками принтеров?

Да, несколько утилит позволяют гибко настраивать принтеры:

  • Default Printer Manager — автоматически меняет принтер по умолчанию в зависимости от сети (например, дома/в офисе).
  • PaperCut — корпоративное решение для управления печалью (включает настройки значков).
  • PrinterLogic — альтернатива CUPS для Linux с расширенными настройками.

Для домашнего использования подойдёт бесплатная утилита DefaultPrinter (доступна на GitHub).

Почему после удаления принтера значок остаётся в трее?

Это может быть:

  1. Остаточный процесс — перезапустите Проводник (explorer.exe) через диспетчер задач.
  2. Кэш значков — очистите кэш командой: 
    ie4uinit.exe -ClearIconCache
  3. Служба печати — перезапустите её: 
    net stop spooler & net start spooler

Если проблема сохраняется, проверьте папку C:\Windows\System32\spool\PRINTERS на наличие застрявших задач.
