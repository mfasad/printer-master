Как установить принтер принтером по умолчанию: инструкции для всех ОС

Принтер — одно из самых востребованных периферийных устройств в офисе и дома. Но что делать, если система упорно отправляет документы на старое устройство, а вам нужно печатать на новом? Или когда после подключения нескольких принтеров компьютер выбирает не тот, который нужен? В этой статье разберём, как правильно установить принтер по умолчанию на разных операционных системах, включая нюансы для Canon, HP, Brother и других популярных брендов.

Проблема выбора принтера по умолчанию кажется простой, но на практике пользователи сталкиваются с массой подводных камней: от сбившихся настроек после обновления Windows до конфликтов драйверов. Мы не только покажем стандартные способы настройки, но и расскажем, как обойти типичные ошибки, когда система игнорирует ваши изменения или принтер «исчезает» из списка устройств. Особое внимание уделим случаям, когда принтер подключён по сети или через USB-хаб — здесь алгоритм действий отличается от классического.

Почему важно настроить принтер по умолчанию

На первый взгляд, выбор принтера по умолчанию — мелочь. Но представьте ситуацию: вы срочно отправляете на печать контракт, а документ уходит на МФУ в другом конце офиса, которое давно не заправляли. Или дома: ребёнок печатает реферат, а файл отправляется на чёрно-белый лазерник вместо цветного Epson EcoTank, из-за чего диаграммы становятся нечитаемыми. Вот почему корректная настройка экономит время и нервы:

  • ⏱️ Скорость работы: не нужно каждый раз выбирать принтер вручную при печати из Word, Excel или браузера.
  • 💰 Экономия расходников: документы не уйдут на устройство с дорогой краской, если вы хотели распечатать на лазернике.
  • 🔄 Автоматизация задач: скрипты, программы бухгалтерии (например, 1С) и облачные сервисы (вроде Google Docs) используют принтер по умолчанию.
  • 🖨️ Сетевая печать: в офисах с несколькими принтерами правильная настройка предотвращает очереди у одного устройства.

Кроме того, некоторые программы (например, Adobe Acrobat или AutoCAD) могут «забывать» последний выбранный принтер и всегда использовать дефолтный. Если его не настроить заранее, придётся тратить время на ручную корректировку перед каждой печатью.

Как установить принтер по умолчанию в Windows 10 и 11

В последних версиях Windows алгоритм выбора принтера по умолчанию изменился. Теперь система может автоматически назначать последний использованный принтер в качестве основного. Это удобно для пользователей с одним устройством, но вызывает путаницу, если принтеров несколько. Разберём оба сценария: когда нужно отключить автоматический выбор и когда требуется вручную задать приоритетное устройство.

Шаг 1: Отключение функции «Управление по умолчанию»

Если Windows постоянно сбрасывает ваши настройки, виновата опция Разрешить Windows управлять принтером по умолчанию. Чтобы её отключить:

  1. Откройте Параметры → Устройства → Принтеры и сканеры (или в Windows 11: Параметры → Bluetooth и устройства → Принтеры и сканеры).
  2. Прокрутите вниз до раздела Связанные параметры и выберите Дополнительные параметры принтера.
  3. Снимите галочку напротив Разрешить Windows управлять принтером по умолчанию.
💡

Если пункта "Дополнительные параметры принтера" нет, попробуйте открыть классическую Панель управления → Устройства и принтеры. В Windows 11 этот путь иногда работает надёжнее.

Шаг 2: Ручной выбор принтера по умолчанию

Теперь, когда автоматическое управление отключено, можно задать приоритетное устройство:

  1. Вернитесь в Принтеры и сканеры.
  2. Найдите в списке нужный принтер (например, Canon PIXMA G3411 или HP LaserJet Pro M404n) и кликните по нему.
  3. Нажмите кнопку Управление (или Открыть очередь в Windows 11).
  4. Выберите Назначить по умолчанию.

Если кнопка Назначить по умолчанию неактивна, проверьте:

  • 🔌 Подключен ли принтер к сети/USB (индикаторы на устройстве должны гореть зелёным).
  • 🖼️ Установлены ли драйверы (в списке не должно быть значка предупреждения рядом с названием).
  • 🔄 Не запущено ли обновление системы (иногда Windows блокирует изменения настроек во время апдейтов).

Решение проблем в Windows

Если принтер не устанавливается по умолчанию или исчезает из списка, попробуйте:

Проблема Причина Решение
Кнопка «Назначить по умолчанию» отсутствует Принтер подключён как сетевой, но драйвер не установлен Установите драйвер с сайта производителя (например, Canon или HP)
Настройки сбрасываются после перезагрузки Конфликт служб печати или вирус Запустите services.msc, найдите службу Диспетчер печати, перезапустите её. Проверьте систему антивирусом
Принтер отображается как «Офлайн» Проблемы с сетевым подключением или USB-портом Проверьте кабель, переподключите USB в другой порт, перезагрузите роутер (для сетевых принтеров)
Ошибка «Не удалось установить принтер по умолчанию» Повреждён реестр или профиль пользователя Создайте нового пользователя в Windows и повторите настройку
⚠️ Внимание: Если вы подключаете принтер через USB-хаб без отдельного питания, устройство может отключаться при высокой нагрузке (например, при печати графики). Используйте хабы с внешним блоком питания или подключайте принтер напрямую к ПК.

Настройка принтера по умолчанию на macOS (Ventura, Sonoma и старше)

На Mac алгоритм проще, чем на Windows, но есть свои нюансы. Например, после обновления системы список принтеров может сбрасываться, а сетевые устройства — «забываться». Разберём, как настроить принтер по умолчанию на macOS Sonoma и более ранних версиях.

Способ 1: Через «Системные настройки»

Самый надёжный метод:

  1. Откройте Системные настройки → Принтеры и сканеры.
  2. В левой части окна выберите нужный принтер (например, Brother DCP-L2550DW).
  3. Кликните правой кнопкой мыши (или удерживая Control) по названию принтера.
  4. Выберите Назначить принтером по умолчанию.

Если принтер подключён по сети, но не отображается в списке, нажмите + внизу окна и выберите Добавить принтер или сканер. Система автоматически найдёт устройства в локальной сети (протокол Bonjour).

Способ 2: Через очередь печати

Альтернативный вариант:

  1. Откройте любой документ (например, в Pages или Preview).
  2. Нажмите Command + P, чтобы открыть диалог печати.
  3. В выпадающем меню Принтер выберите нужное устройство.
  4. Нажмите Настройки принтера... и установите галочку Использовать этот принтер по умолчанию.
⚠️ Внимание: На macOS сетевые принтеры с нестандартными протоколами (например, Kyocera с LPD) могут требовать ручного добавления через IP-адрес. Уточните протокол в инструкции к принтеру.

Решение проблем на Mac

Типичные ошибки и их исправление:

  • 🔄 Принтер исчезает после сна: Отключите режим энергосбережения для USB-портов в Системные настройки → Экономия энергии.
  • 🖼️ Драйвер не устанавливается: Скачайте ПО с сайта производителя (например, Brother), а не полагайтесь на стандартные драйверы macOS.
  • 🌐 Не видно сетевой принтер: Проверьте, что компьютер и принтер находятся в одной подсети (например, оба на 192.168.1.x).
Как узнать IP-адрес принтера для ручного подключения?

1. На принтере нажмите кнопку Menu (или Settings).

2. Перейдите в раздел Network или Сеть.

3. Найдите пункт TCP/IP, Wi-Fi Direct или IP Address.

4. Запишите адрес (например, 192.168.1.105) и используйте его при добавлении принтера в macOS.

Установка принтера по умолчанию в Linux (Ubuntu, Mint, Fedora)

В Linux настройка принтера по умолчанию зависит от используемой графической оболочки (GNOME, KDE, XFCE) и системы печати (CUPS). Мы рассмотрим универсальный способ через CUPS, который работает на большинстве дистрибутивов, включая Ubuntu 22.04 и Fedora 38.

Способ 1: Через веб-интерфейс CUPS

CUPS (Common Unix Printing System) предоставляет веб-панель для управления принтерами:

  1. Откройте браузер и перейдите по адресу: http://localhost:631.
  2. Перейдите на вкладку Printers.
  3. Найдите в списке нужный принтер (например, Xerox Phaser 3020) и кликните по нему.
  4. В разделе Administration выберите Set As Server Default.
  5. Введите логин и пароль администратора (обычно это данные вашей учётной записи в Linux).

Если страница localhost:631 не открывается, убедитесь, что служба CUPS запущена:

sudo systemctl status cups

Если служба остановлена, запустите её:

sudo systemctl start cups

Способ 2: Через терминал

Для опытных пользователей подойдёт командный метод:

  1. Получите список всех принтеров:
lpstat -p -d
  1. Установите принтер по умолчанию (замените printer_name на реальное имя, например, HP_LaserJet_Pro_M402):
lpoptions -d printer_name

Чтобы изменения сохранились после перезагрузки, добавьте команду в автозагрузку или отредактируйте файл ~/.cups/lpoptions.

Решение проблем в Linux

Частые ошибки и их исправление:

  • 🐧 Принтер не обнаруживается: Установите пакет hplip для HP или foomatic-db для других брендов: 
    sudo apt install hplip
  • 🔌 USB-принтер не определяется: Проверьте права доступа к устройству: 
    lsusb

    Если принтер виден, но не работает, добавьте пользователя в группу lp:

    sudo usermod -aG lp $USER
  • 🌐 Сетевой принтер недоступен: Откройте порты 631 и 515 в брандмауэре: 
    sudo ufw allow 631
⚠️ Внимание: В некоторых дистрибутивах (например, Arch Linux) служба CUPS не запускается автоматически. Чтобы включить автозагрузку, выполните: 
sudo systemctl enable --now cups
💡

В Linux сетевые принтеры часто требуют ручной установки PPD-файла (драйвера). Скачайте его с сайта производителя и загрузите через веб-интерфейс CUPS при добавлении устройства.

Особенности для сетевых и USB-принтеров

Способ подключения принтера напрямую влияет на то, как система с ним взаимодействует. Разберём ключевые различия между USB, Wi-Fi и Ethernet-подключениями, а также типичные ошибки при настройке.

USB-принтеры: почему они «исчезают» из списка

Принтеры, подключённые по USB, чаще других теряют статус «по умолчанию» из-за:

  • 🔌 Нестабильного питания: Если принтер подключён через хаб без внешнего питания, при печати больших документов он может отключаться.
  • 🔄 Конфликтов портов: Windows иногда «забывает» принтер, если USB-порт был деактивирован для экономии энергии.
  • 🖥️ Сбоев драйверов: После обновления ОС старые драйверы могут перестать работать.

Решения:

  • Подключайте принтер напрямую к USB-порту на задней панели ПК (они питаются стабильнее, чем передние).
  • Отключите экономию энергии для USB в Диспетчере устройств:
    1. Откройте Диспетчер устройств (Win + X → Диспетчер устройств).
    2. Разверните раздел Контроллеры USB.
    3. Найдите Корневой USB-концентратор, кликните правой кнопкой → Свойства → Управление электропитанием.
    4. Снимите галочку Разрешить отключение этого устройства для экономии энергии.
  • Переустановите драйвер принтера вручную (скачайте с официального сайта, а не через Windows Update).

    • Сетевые принтеры: настройка приоритета

    Сетевые принтеры (подключённые по Wi-Fi или Ethernet) имеют свои нюансы:

    • 📡 Wi-Fi-принтеры могут «отваливаться» при слабом сигнале или смене канала роутера.
    • 🔌 Ethernet-принтеры требуют статического IP-адреса, иначе после перезагрузки роутера они могут стать недоступны.
    • 🖨️ Общие принтеры (расшаренные через другой ПК) зависят от состояния хост-компьютера.

      • Чтобы сетевой принтер оставался приоритетным:

      1. Назначьте ему статический IP в настройках роутера (через привязку по MAC-адресу).
      2. В настройках принтера отключите режим сна (Sleep Mode), иначе устройство будет «пропадать» из сети.
      3. Если принтер расшарен с другого ПК, убедитесь, что хост-компьютер не уходит в спящий режим.
    Тип подключения Типичная проблема Решение
    USB Принтер исчезает после сна ПК Отключите экономию энергии для USB-портов в Диспетчере устройств
    Wi-Fi Принтер не отвечает Переподключите принтер к сети, проверьте канал Wi-Fi (используйте 5 GHz для стабильности)
    Ethernet IP-адрес принтера изменился Настройте статический IP в роутере или на самом принтере
    Общий (расшаренный) Принтер недоступен Проверьте, что хост-ПК включён и не блокирует доступ брандмауэром

    Как проверить, что принтер установлен по умолчанию

    После настройки важно убедиться, что система действительно использует выбранный принтер. Вот как это сделать на разных платформах:

    В Windows

    Способы проверки:

    1. Откройте Параметры → Устройства → Принтеры и сканеры. Рядом с принтером по умолчанию будет надпись Готов (по умолчанию).
    2. Отправьте тестовый документ на печать (например, из Блокнота). В окне печати должен быть выбран нужный принтер.
    3. Запустите команду в PowerShell: 
      Get-Printer | Where-Object {$_.Default} | Select-Object Name

      Она выведет имя принтера по умолчанию.

На macOS

Проверка:

  1. Откройте Системные настройки → Принтеры и сканеры. Принтер по умолчанию будет выделен жирным шрифтом.
  2. Откройте любой документ и нажмите Command + P. В поле Принтер должно отображаться нужное устройство.
  3. В терминале выполните: 
    lpstat -d

    Команда покажет текущий принтер по умолчанию.

    4. В Linux

    Проверка через терминал:

    lpstat -d

    Или через веб-интерфейс CUPS (http://localhost:631), где рядом с принтером по умолчанию будет звёздочка (*).

    Если тестовая печать идёт не на тот принтер, проверьте:

    • Не сбились ли настройки после перезагрузки (особенно актуально для Linux).
    • Не конфликтуют ли драйверы (например, если установлены и стандартные драйверы macOS, и производителя).
    • Не блокирует ли печать антивирус или брандмауэр (временно отключите их для теста).

    Частые ошибки и их решения

    Даже после правильной настройки принтер по умолчанию может работать некорректно. Разберём типичные ошибки и способы их исправления.

    Ошибка: «Не удалось установить принтер по умолчанию» (Windows)

    Причины и решения:

    • 🔧 Повреждён реестр: Запустите regedit, перейдите в HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\Windows и удалите параметр LegacyDefaultPrinterMode. Перезагрузите ПК.
    • 👤 Ограничения групповой политики: В корпоративных сетях администратор может блокировать изменение принтера по умолчанию. Обратитесь в IT-службу.
    • 🖼️ Конфликт драйверов: Удалите все принтеры в Устройства и принтеры, перезагрузите ПК и добавьте их заново.

    Ошибка: Принтер по умолчанию сбрасывается после обновления

    Это частая проблема после крупных апдейтов Windows (например, переход с Windows 10 на 11). Решения:

    • Перед обновлением запишите имя принтера по умолчанию и после апдейта назначьте его заново.
    • Если принтер исчез из списка, удалите его и добавьте заново через Добавить принтер или сканер.
    • Для сетевых принтеров проверьте, не изменился ли их IP-адрес после перезагрузки роутера.

Ошибка: macOS не сохраняет принтер по умолчанию

В macOS иногда сбиваются настройки из-за:

  • 🔄 Обновления системы: После апдейта перезагрузите компьютер и назначьте принтер заново.
  • 🖼️ Конфликта драйверов: Удалите принтер, скачайте свежий драйвер с сайта производителя и добавьте устройство заново.
  • 📡 Проблем с Bonjour: Перезапустите службу: 
    sudo launchctl stop com.apple.mDNSResponder

    sudo launchctl start com.apple.mDNSResponder

    • Ошибка: Linux не видит принтер по умолчанию

    В Linux проблемы часто связаны с:

    • 🐧 Отсутствием прав: Добавьте пользователя в группу lpadmin: 
      sudo usermod -aG lpadmin $USER
    • 🖨️ Неправильным PPD-файлом: Удалите принтер через CUPS и добавьте заново, выбрав правильный драйвер.
    • 🌐 Блокировкой портов: Проверьте, что порты 631 и 515 открыты: 
      sudo netstat -tulnp | grep cups
    ⚠️ Внимание: Если вы используете виртуальные машины (например, VirtualBox или VMware), принтер по умолчанию может не сохраняться из-за особенностей передачи USB-устройств. В этом случае настройте принтер непосредственно на хост-системе и используйте общий доступ к нему из виртуальной машины.

    FAQ: Ответы на частые вопросы

    Можно ли сделать принтер по умолчанию для конкретной программы (например, только для Excel)?

    Да, некоторые программы (например, Microsoft Office или Adobe Acrobat) позволяют задать принтер по умолчанию отдельно для себя. Для этого:

    1. Откройте документ в программе.
    2. Нажмите Ctrl + P (или Command + P на Mac).
    3. Выберите нужный принтер в выпадающем меню.
    4. Найдите опцию вроде Сохранить как принтер по умолчанию или Использовать всегда (в Word это может быть в настройках печати).

    Обратите внимание: это работает только для текущей программы и не влияет на системные настройки.

    Почему после перезагрузки принтер по умолчанию сбрасывается на «Microsoft XPS Document Writer»?

    Эта проблема типична для Windows и связана с:

    • Включённой опцией Разрешить Windows управлять принтером по умолчанию (отключите её, как описано выше).
    • Повреждением системного файла Devices.xml. Чтобы исправить:
      1. Откройте Проводник и включите показ скрытых файлов (Вид → Скрытые элементы).
      2. Перейдите по пути: C:\Users\Ваше_имя_пользователя\AppData\Local\Microsoft\Windows\ConnectedDevices.
      3. Удалите файл Devices.xml.
      4. Перезагрузите ПК и назначьте принтер по умолчанию заново.
    Как сделать принтер по умолчанию для всех пользователей на одном ПК?

    В Windows и Linux можно задать принтер по умолчанию глобально:

    В Windows:

    1. Откройте Панель управления → Администрирование → Управление печатью.
    2. Кликните правой кнопкой по принтеру и выберите Назначить принтером по умолчанию.

    В Linux (через CUPS):

    1. Откройте http://localhost:631.
    2. Перейдите в Administration → Set Default Options.
    3. Выберите принтер и примените настройки ко всем пользователям.
