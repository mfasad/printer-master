Как переключить принтер на другой: полная инструкция для всех ОС

Переключение принтера на другой — задача, с которой сталкиваются как офисные сотрудники, так и домашние пользователи. Некорректно выбранное устройство печати может привести к потере времени, ошибкам при выводе документов или даже сбоям в работе программ. Особенно актуальна эта проблема в сетях с несколькими МФУ или при использовании виртуальных принтеров (например, для создания PDF).

В этой статье мы разберём все способы смены принтера по умолчанию — от стандартных настроек Windows до тонкостей работы с macOS и Linux. Отдельное внимание уделим сетевым устройствам, которые часто требуют дополнительной настройки драйверов. Если вы работаете с 1С, AutoCAD или другими специализированными программами, где принтер выбирается отдельно — здесь тоже найдёте решение.

Прежде чем приступать, убедитесь, что новый принтер подключён к компьютеру или сети и распознаётся системой. Если устройство не отображается в списке, сначала установите драйверы с официального сайта производителя (HP, Canon, Brother и др.).

1. Как изменить принтер по умолчанию в Windows 10/11

В современных версиях Windows процесс смены принтера упрощён, но есть нюансы для разных сборок ОС. Рассмотрим универсальный метод, который работает и для Windows 10 22H2, и для Windows 11 24H2.

Откройте Параметры → Устройства → Принтеры и сканеры. Здесь вы увидите список всех установленных устройств. Найдите нужный принтер и нажмите на него. Появится кнопка Управление — кликните на неё, затем выберите Установить по умолчанию.

Убедиться, что принтер включён и подключён к сети/компьютеру|

Проверить наличие бумаги и чернил/тонера|

Обновить драйверы через Диспетчер устройств|

Перезагрузить компьютер после установки нового принтера-->

Если кнопка Установить по умолчанию неактивна, это означает, что в системе включена функция "Разрешить Windows управлять принтером по умолчанию". Чтобы отключить её:

  1. Перейдите в Параметры → Устройства → Принтеры и сканеры.
  2. В верхней части окна снимите флажок напротив Разрешить Windows управлять принтером по умолчанию.
  3. После этого кнопка станет активной.
⚠️ Внимание: В корпоративных сетях настройка принтера по умолчанию может блокироваться групповой политикой. Если вы не видите нужного устройства или не можете его выбрать, обратитесь к администратору IT-отдела.

Для быстрого доступа к настройкам принтера можно использовать команду control printers в окне Выполнить (Win + R). Это откроет классическую панель управления, где также доступен выбор устройства по умолчанию.

2. Смена принтера в macOS (Ventura, Sonoma, Sequoia)

На Mac процесс переключения принтера интегрирован в систему печати CUPS (Common Unix Printing System). Интерфейс здесь более минималистичный, но есть свои особенности.

Откройте Системные настройки → Принтеры и сканеры. В левой части окна выберите нужный принтер, затем нажмите Параметры и материалы. В выпадающем меню Принтер по умолчанию укажите требуемое устройство.

  • 🖨️ Локальный принтер: Подключён напрямую через USB или Thunderbolt. В списке отображается с пометкой USB.
  • 🌐 Сетевой принтер: Подключён по Wi-Fi, Ethernet или через AirPrint. Может иметь имя вида HP_OfficeJet_Pro_9015_[IP].
  • 📄 Виртуальный принтер: Например, Save as PDF или Microsoft XPS Document Writer.

Если принтер не отображается, нажмите + в нижней части окна и добавьте его вручную. Для сетевых устройств может потребоваться ввод IP-адреса или выбора протокола (LPD/LPR, IPP, SMB).

⚠️ Внимание: В macOS Sequoia (2026+) изменился механизм работы с драйверами сторонних производителей. Если после обновления системы принтер перестал определяться, установите последнюю версию ПО с сайта производителя.

Для проверки текущего принтера по умолчанию можно использовать терминал. Введите команду:

lpstat -d

Чтобы сменить устройство, выполните:

lpoptions -d PRINTER_NAME

где PRINTER_NAME — точное имя принтера из списка (узнать его можно командой lpstat -a).

3. Настройка принтера по умолчанию в Linux (Ubuntu, Debian, Fedora)

В дистрибутивах на базе Linux управление печатью осуществляется через CUPS (как и в macOS), но интерфейс может отличаться. Рассмотрим два способа: через графический интерфейс и терминал.

Для графической настройки откройте Настройки → Устройства → Принтеры (в GNOME) или Системные настройки → Печать (в KDE). Найдите нужный принтер, кликните правой кнопкой и выберите Установить как принтер по умолчанию.

Если графического интерфейса нет (например, на сервере), используйте команды:

lpoptions -d PRINTER_NAME # Установить принтер по умолчанию

lpstat -d # Просмотреть текущий принтер

Для добавления нового принтера в CUPS:

  1. Откройте в браузере http://localhost:631 (веб-интерфейс CUPS).
  2. Перейдите в раздел Administration → Add Printer.
  3. Выберите устройство из списка или укажите его IP-адрес для сетевого принтера.
  4. Установите драйвер (для большинства моделей HP, Canon, Brother подойдут стандартные PPD-файлы).
Дистрибутив Команда для установки CUPS Путь к конфигу
Ubuntu/Debian sudo apt install cups /etc/cups/cupsd.conf
Fedora/RHEL sudo dnf install cups /etc/cups/cupsd.conf
Arch Linux sudo pacman -S cups /etc/cups/cupsd.conf
OpenSUSE sudo zypper install cups /etc/cups/cupsd.conf

Если принтер не определяется, проверьте, запущена ли служба CUPS:

sudo systemctl status cups

При необходимости перезапустите её:

sudo systemctl restart cups

4. Переключение принтера в специализированных программах

Некоторые приложения (например, 1С:Предприятие, AutoCAD, Adobe Photoshop) игнорируют системные настройки принтера и требуют отдельной настройки. Рассмотрим наиболее распространённые случаи.

В 1С:Предприятие 8.3 выбор принтера осуществляется непосредственно при печати документа:

  1. Откройте нужный документ (например, счёт или накладную).
  2. Нажмите Печать или используйте комбинацию Ctrl+P.
  3. В окне печати выберите требуемый принтер из выпадающего списка.
  4. При необходимости настройте параметры страницы (Ориентация, Масштаб, Поля).

В AutoCAD принтер выбирается в диалоговом окне Plot (Ctrl+P):

  • 📌 В разделе Printer/plotter укажите имя устройства.
  • 📏 В Paper size выберите формат листа (например, A3 или ANSI B).
  • 🎨 В Plot style table настройте цветовую схему (monochrome.ctb для ч/б печати).

В Adobe Photoshop и Illustrator принтер выбирается в меню Файл → Печать (Ctrl+P). Здесь также доступны расширенные настройки цветопередачи (Цветовой профиль) и качества (DPI).

⚠️ Внимание: В 1С настройки принтера могут сбрасываться при обновлении конфигурации. Если после апдейта документ печатается не на том устройстве, проверьте параметры в разделе Администрирование → Печать.

Для Microsoft Office (Word, Excel) принтер по умолчанию берётся из системных настроек, но его можно переопределить непосредственно перед печатью. Откройте файл, нажмите Ctrl+P, затем выберите нужное устройство в выпадающем меню.

5. Решение проблем при смене принтера

Если принтер не переключается или возникают ошибки, проверьте следующие моменты:

  • 🔌 Подключение: Убедитесь, что кабель USB или сетевое соединение стабильно. Для Wi-Fi-принтеров проверьте сигнал роутера.
  • 🖥️ Драйверы: Удалите старые драйверы через Диспетчер устройств и установите актуальные с сайта производителя.
  • 📡 Сетевые настройки: Для сетевых принтеров проверьте, что IP-адрес устройства не конфликтует с другими гаджетами в сети.
  • 🔄 Очередь печати: Откройте Панель управления → Устройства и принтеры, найдите проблемный принтер, кликните Просмотр очереди печати и очистите её.

Частая ошибка в Windows — 0x0000007c или 0x00000057 при попытке сменить принтер. Она возникает из-за повреждения системных файлов. Чтобы исправить:

  1. Откройте Командную строку от имени администратора.
  2. Выполните команды: 
    sfc /scannow

    dism /online /cleanup-image /restorehealth

  3. Перезагрузите компьютер.

В macOS ошибка "Принтер не отвечает" часто связана с блокировкой порта 631 (используется CUPS). Чтобы разблокировать:

sudo lsof -i:631

sudo kill -9 PID

где PID — идентификатор процесса из первой команды.

Что делать, если принтер исчез из списка после обновления Windows?

После крупных обновлений (например, с Windows 10 на 11) система может сбросить драйверы принтера. В этом случае:

1. Отключите принтер от сети/USB.

2. Удалите его через Параметры → Устройства → Принтеры и сканеры.

3. Перезагрузите компьютер.

4. Подключите принтер заново — Windows должна автоматически установить драйвер.

Если устройство не определилось, скачайте драйвер с сайта производителя и установите его вручную.

Для сетевых принтеров HP (например, HP LaserJet Pro M404n) может потребоваться ручная настройка IP-адреса. Зайдите в веб-интерфейс принтера (обычно по адресу http://[IP_принтера]), перейдите в раздел Network и проверьте, что DHCP включён или задан статический IP.

6. Настройка принтера по умолчанию для конкретных пользователей

В корпоративных средах или наемых компьютерах может потребоваться установить разные принтеры по умолчанию для разных пользователей. Это реализуется через групповые политики (в Windows) или настройки CUPS (в Linux/macOS).

В Windows для этого:

  1. Откройте Редактор локальной групповой политики (gpedit.msc).
  2. Перейдите по пути: Конфигурация пользователя → Административные шаблоны → Панель управления → Принтеры.
  3. Активируйте политику Разрешить пользователям выбирать принтер по умолчанию.

В Linux для каждого пользователя можно создать отдельный ~/.cups/lpoptions файл с настройками. Например:

Default PRINTER_NAME

Destination PRINTER_NAME

В macOS настройки принтера по умолчанию привязываются к учётной записи. Чтобы сбросить их, удалите файл:

rm ~/Library/Preferences/org.cups.PrintingPrefs.plist
ОС Метод настройки Применяется ко всем пользователям?
Windows Групповые политики (gpedit.msc) Нет (настраивается отдельно)
macOS Файл ~/Library/Preferences/... Нет
Linux Файл ~/.cups/lpoptions Нет
Windows Server Политика Print Management Да (глобально)

Если вы используете Terminal Server или Citrix, настройка принтера по умолчанию может осуществляться через политики сессии. В этом случае изменения применяются только на время подключения пользователя.

7. Автоматизация смены принтера с помощью скриптов

Для частых переключений между принтерами (например, в офисе с несколькими МФУ) удобно использовать скрипты. Рассмотрим примеры для разных ОС.

В Windows можно создать .bat-файл:

@echo off

rundll32 printui.dll,PrintUIEntry /y /n"PRINTER_NAME"

Сохраните файл с расширением .bat и запускайте его, когда нужно сменить принтер.

В macOS/Linux используйте bash-скрипт:

#!/bin/bash

lpoptions -d PRINTER_NAME

Сделайте файл исполняемым:

chmod +x switch_printer.sh

Для PowerShell (Windows) подойдёт такой скрипт:

$printer = Get-WmiObject -Query"SELECT * FROM Win32_Printer WHERE Default=$true"

$printer.SetDefaultPrinter("PRINTER_NAME")

Скрипты можно интегрировать в планировщик задач (например, автоматически переключать принтер в начале рабочего дня) или привязать к горячим клавишам через AutoHotkey (Windows) или Karabiner (macOS).

💡

Если вам часто приходится переключаться между принтерами, создайте ярлыки для каждого скрипта и разместите их на рабочем столе. Так вы сможете менять устройство печати в один клик.

Для корпоративного использования можно развернуть скрипт через Group Policy (Windows) или Ansible (Linux/macOS), чтобы все сотрудники получали актуальные настройки автоматически.

8. Особенности работы с сетевыми и облачными принтерами

Сетевые и облачные принтеры (например, Google Cloud Print, HP ePrint) требуют дополнительной настройки. Рассмотрим ключевые моменты.

Для подключения сетевого принтера по IP-адресу:

  1. Узнайте IP принтера (обычно указан на наклейке или в меню устройства).
  2. В настройках печати добавьте новый принтер, выбрав опцию Добавить по TCP/IP-адресу.
  3. Укажите IP, порт (9100 для RAW, 631 для IPP) и модель.

Для Google Cloud Print (до 2021 года) или его аналогов:

  • 🌍 Подключите принтер к аккаунту Google через веб-интерфейс.
  • 📱 Установите расширение Google Cloud Print в Chrome (если ещё доступно).
  • 🔄 В настройках печати выберите Save to Google Drive или другой облачный принтер.

Для принтеров HP с поддержкой ePrint:

  1. Зарегистрируйте устройство на портале hpconnected.com.
  2. Активируйте функцию Web Services в меню принтера.
  3. Отправляйте документы на печать по email вида printer_name@hpeprint.com.
⚠️ Внимание: С 2021 года Google Cloud Print официально закрыт. Альтернативы: HP Smart, Epson Connect, Canon PRINT или локальные решения вроде CUPS с удалённым доступом.

Для печатных серверов (например, на базе Windows Server или CUPS):

  • 🖥️ На сервере добавьте принтер и опубликуйте его в сети.
  • 🔗 На клиентских машинах подключитесь к серверу через \\server_name\printer_share (Windows) или smb://server_name/printer_share (macOS/Linux).

Если ваш сетевой принтер поддерживает протокол IPP (Internet Printing Protocol), используйте адрес вида http://[IP_принтера]:631/printers/[имя_принтера] для подключения. Этот метод работает в большинстве современных ОС без дополнительных драйверов.

💡

Для стабильной работы сетевого принтера назначьте ему статический IP-адрес в настройках роутера или непосредственно на устройстве. Это предотвратит потерю соединения после перезагрузки маршрутизатора.

Если принтер подключён через USB к другому компьютеру в сети, используйте функцию общего доступа (Windows) или Samba (Linux/macOS). В этом случае основной компьютер должен быть включён для печати.

FAQ: Частые вопросы по смене принтера

Можно ли сделать разные принтеры по умолчанию для разных программ?

Да, но это зависит от программы. Например, в Microsoft Office принтер выбирается при печати, а в 1С можно настроить отдельные шаблоны печати для разных устройств. Универсального системного решения нет — каждый софт управляет печатью самостоятельно.

Почему после обновления Windows принтер сбросился на"Microsoft XPS Document Writer"?

Это стандартное поведение Windows при сбое драйверов. Чтобы исправить:

  1. Удалите проблемный принтер через Параметры → Устройства.
  2. Подключите его заново (система должна установить драйвер автоматически).
  3. Если не помогло, скачайте драйвер с сайта производителя и установите вручную.

Как переключить принтер на Android или iOS?

На мобильных устройствах принтер выбирается непосредственно в приложении печати (например, Google Cloud Print, HP Smart, Canon PRINT). Системного"принтера по умолчанию" в Android или iOS нет — каждый раз приходится выбирать устройство вручную.

Можно ли изменить принтер по умолчанию через реестр Windows?

Да, но это не рекомендуется без опыта. Ключ находится по пути: HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\Windows\Device.

Изменение этого параметра может привести к сбоям в работе системы печати.

Почему принтер по умолчанию сбрасывается после перезагрузки?

Это может быть вызвано:

  • Функцией Разрешить Windows управлять принтером (отключите её в параметрах).
  • Конфликтом драйверов (переустановите ПО для принтера).
  • Групповой политикой (в корпоративных сетях).
